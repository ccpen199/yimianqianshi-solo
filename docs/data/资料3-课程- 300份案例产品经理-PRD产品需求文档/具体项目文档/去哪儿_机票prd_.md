4.项目方案概述 
4.1产品方案概述  
从国内、国际·港澳台、单程、往返四个较大维度供用户选择想购买的机票。另外还有
出发地点、到达地点、出发日期、返程日期等具体条件供用户筛选机票。  
4.2技术方案概述  
本版块涉及到用户状态判断、订单历史、出票状态、当下热门旅游地、机场归属国、国
家拥有的机场、当前日期调取、实时国内国际航班数量等内容。  
4.3功能点介绍  
主要功能，根据用户所填写条件筛选符合用户条件的机票信息。
