2. 产品概况
2.1 行业分析
职场新锐四处迁徙的短租生活，可以用外卖代替热水壶煮泡面，但是鲜有房东留下洗衣机
、空气净化器、净水机等品质家电，冰箱、空调、热水器都是老旧的款式。
而如果自行购入，价格低廉的无法满足追求精致生活的都市人的需求，而价格昂贵的在短
租生活中又显得不够使用，并且一旦离开这座城市时后续的处理也是一个很大的问题。
这一情况为家电租赁市场打开了大门，为了解决短租一族家电使用的问题，同时借助家电
企业的优势，设想建立“家电二手生态圈”的可行性，提供家电产品的租借以及流通平台
，方便客户进行家电的租借，转租以及购买等。
2.2 目标人群
在北上广深等一二线城市工作的租房一族，对生活质量有一定需求，但由于仍处于职业探
索期，工作地点等尚不稳定，具有一定消费能力的人以及有一定事业。但由于大城市房价
高昂，仍需要租房居住的具有品牌意识、消费意识、享受生活的人群。

2.5 产品介绍
海尔提供家电单件或组合租赁（经济型和高端型），不同的组合方式有相应的折扣，需要
进行“海电”的注册以及实名认证，缴纳一定的押金。 海尔提供家电上门安装服务，负
责后期的维护，收取服务费。 租期的选择分为3个月（进行短期实习的学生党），1年
（应届毕业生），3~5年（有一定事业基础的青年）。 到期后可续借，商品损耗度符合
退还要求可退换押金，海尔负责家电的回收，回收家电进行维护后以折扣的价格继续加入
“海电”进行租赁；可提供“租赁转购买”，对于租赁超过一定年限的家电可赠送给租户
（成功回收成本）。 针对提前退租的情况，提供两种选择：一，根据契约精神，提前退
还需要扣除一定的押金；二，自行利用海电平台提供的二手转租模式，将家电在剩余的时


间租给别人，双方达成协议后，家电转租至新的租户名下，租金由双方自行协商，但是上
缴至海电平台的租金应为官方对于相应损耗的家电的租金定价。
2.6 产品定位
属于购物类APP，需求驱动产品。深入挖掘短租一族的家电需求，提供租借、转租、退租
的一条龙家电租赁相关服务，打造二手家电短租生态圈，联系线上选购与线下提货，安装
维修为一体。

4. 全局说明
4.1 功能权限
4.1.1 未登录状态/已登录状态
已登录状态可执行所有操作； 未登录状态下只能浏览界面和商品详情。
4.1.2 实名认证/未实名认证
未实名认证不可进行租借，实名认证可以进行租借，退租和转租等一系列操作。
4.2 键盘/提示信息交互说明
点击手机注册/手机登录输入框时弹出数字键盘； 点击其他输入框弹出字母键盘。
5. 产品功能
5.1 启动页
用户场景：用户打开APP的欢迎界面；
功能描述：欢迎界面，点击进入下一界面；
优先级：高。

说明：
用户首次打开App，进入欢迎页面； 非首次打开App进入启动页，一般设置为家电产品
广告页面，对广告页不进行任何点击，2s后自动进入App首页；点击跳过，直接进入
App首页；点击广告页图片没有热区的地方，进入广告页；点击详情，进入产品宣传文
章页面。
5.2 注册/登录
用户场景：用户进入APP后在登录界面点击注册或进行登录；
功能描述：用户注册，再次登录；
优先级：高；
输入/前置条件：打开APP进行注册操作。

说明：
键盘说明：点击输入框时，弹出键盘。 按钮说明：手机号输入框有字符时，发送验证码
按钮点亮，可点击，否则变灰，不可点击；所有输入框有字符时，注册按钮点亮可点击，
否则变灰，不可点击；点击发送验证码后，按钮文本信息倒计时60秒开始，不可点击；
倒计时到第0秒时，按钮文本信息改变“再次发送”，可点击；注册成功后，跳转至登录
页。 当文本框没有字符的时候，登录按钮不可点击，灰色状态，当账号密码输入后，按
钮亮起，可以点击；登录成功后，进入操作台，否则toast提示“账号或密码错误”。


5.3 首页
用户场景：浏览APP首页；
功能描述：浏览活动，浏览推荐；
优先级：中；
输入/前置条件：打开APP；
用户界面：

说明：
左右划动可切换BANNER页面。点击BANNER页面可至相应活动界面。 常用功能按钮
，点击可快速到达各功能页面，同时当前所在功能界面字体呈现绿色。 展示不同的家电
分类，点击对应分类进入浏览相应家电页面。 根据兴趣进行推荐商品的展示，点击图片
可进入商品详情界面。
5.4 搜索及筛选
用户场景：在APP内搜索家电产品，根据条件进行筛选；
功能描述：搜索家电产品，并根据要求进行排序，同时可以根据选定条件进行筛选；
优先级：高；
输入/前置条件：打开APP；
用户界面：

说明：
顶部搜索输入框，搜索栏提供关键词搜索方式，点击输入框跳出键盘。 搜索完成后页面
转至搜索的家电产品排列界面。 按新旧顺序、价格高低、距离远近对展示的家电进行排
序。 点击右上角图标可出现筛选项，可根据新旧程度、组合搭配、区域、价格区间四种
条件对家电产品进行筛选，这四种条件采用下拉框的方式进行选择。 产品展示区域，点
击图片或产品名称跳转到产品详情页面。 点击搜索输入框弹出字母键盘。
5.5 家电查看与租借
用户场景：查看家电产品详情，进行家电租借；


功能描述：查看家具的图片，各种参数，以及其他说明和详情，查看家电当前所在地址，
查看家电租金，与客服或转租者进行聊天沟通，进行租借；
优先级：高；
输入/前置条件：打开APP；
用户界面：


说明：
点击产品图片或名称进入产品详情页面。 左右滑动切换产品图片。 详情界面显示家电名
称、家电当前所在地址。 下拉框选择不同的租借天数，转租价格相应变化。 点击收藏，
该产品收藏至“我的收藏”，在设置界面的“我的收藏”中可以查看。 点击与租主聊天
，可以与直租的客服或转租的租主进行沟通了解。 点击租借跳转至确认租借界面。 确认
租借界面显示租借商品名称，可选择租借的日期，租金会随之变动。 确认租借按钮显示
所需付款的总金额，点击确认租借跳转至外部支付宝等付款链接（跳过），付款成功回到
上一界面。若放弃租借，左滑返回至商品详情界面。
5.6 租借管理
用户场景：对已经进行的家电进行管理；
功能描述：分类查看进行租借的家电，并且对家电进行不同的操作，包括续租、退租、转
租等；
优先级：高；
输入/前置条件：打开APP；
用户界面：

说明：


点击下方功能栏的我的，跳转至目前租借的家电界面。 该界面按照目前家电的情况划分
为不同的分类，下方空白区域可以进行商品推介的展示。 点击不同的分类，进入该分类
的具体界面，该界面显示当前在该状态的家电。 点解某家电的照片或名称进入该家电当
前租借情况详情界面。 租借情况详情界面显示家电名称、租借起始日期和到期日期，剩
余租借天数。 租借情况详情界面下方有三个按钮，分别可以进行续租、退租、转租三种
操作。

说明：
点击续租进入该界面，该界面显示续租商品名称、续租日期、金价格。 续租日期可以通
过下拉框进行选择，价格随着日期的选择有相应的价格租金的付款方式：有一次付清和每
月付两种形式。 确认租借按钮显示所需付款的总金额，点击确认租借跳转至外部支付宝
等付款链接。 点击确认租借跳转至外部支付宝等付款链接（跳过），付款成功回到上一
界面。若放弃转租，左滑返回至上一界面。

说明：
点击退租进入该界面，该界面显示退租商品名称，距离租期结束的天数，违约金价格。 
用小号字符显示转租字样，点击转租可跳转至转租界面。 确认退租按钮上显示所需付款
的总金额，下方显示退租金额的返回道路，该返回金额为预先存的租金减去违约金的总金
额。 确定退租界面点击确定退租按钮可返回上一界面，若放弃退租，左滑返回至上一界
面。
说明：
点击转租进入该界面，该界面显示商品名称、转租日期、转租价格设置、添加商品详情。 
转租日期可通过下拉框进行设置。 点击转租价格设置跳出键盘，下滑可收起键盘返回该
界面。 点击添加商品详情跳转至添加详情界面，该界面可以添加文字描述和添加照片。 
在添加详情界面点击描述内容框，跳出键盘，下滑可收起键盘。 在添加详情界面点击加
号出现下拉菜单，可以选择手机拍照或相册选择两种添加照片的方式。 点击手机拍照跳
转至照相机拍摄界面，下滑收起照相机界面返回添加详情界面。 点击相册选择跳转至相


片选择界面，在相片选择界面选择右上角取消按钮返回添加详情界面。 编辑完成后，点
击添加详情界面，确定按钮可返回转租界面。若想放弃编辑，点击添加详情界面左上角返
回按钮，可返回转租界面。 确定转租界面点击确定转租按钮可返回上一界面，若放弃转
租，左滑返回至上一界面。

6. 异常说明
6.1 网络异常
说明：
在登录时网络有异常时，会停留在主界面，并显示当前网络异常。 在用户使用App过程
中，突发网络中断时会相应的弹出Toast提醒用户“似乎已断开与互联网的连接”。


6.2 数据丢失
除了网络异常及登录异常以外，用户还可能遇到在使用App过程中程序突然崩溃等异常
状况，这些状况都会造成一定程度上的数据丢失。
在使用App时接打电话，程序会自动设置为后台运行，但若在后台运行的时间过长，再
次打开程序页面会默认转至“欢迎引导页”。而在手机关机和App崩溃的情况下，App
只能重新打开，而不能回到之前操作的页面。 此外App对支付订单功能设置了十五分钟
支付时效，这也在一定程度上能够解决可能的数据丢失造成的支付问题，未支付的订单可
至“我的”中相应的租借种类中寻找。
