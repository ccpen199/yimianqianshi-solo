3.1 登录页面
登录状态下可以进行所有操作。
未登录状态下，不能收藏商品，不可参与任何形式的购买（单独或拼单），不可与客服沟
通，不可查看关注信和个人信息，要进行上述操作必须跳转登录页面。


（1）页面名称：登陆页面
（2）页面入口：未登录状态下，当用户点击待付款、待分享、待发货、待收货、待评价
、优惠券、商品收藏、店铺关注、历史浏览、退款售后、多多果园、1分抽奖、天天领现
金、多多爱消除、收获地址、我的评价、官方客服、设置等界面入口，或者点击客服、收
藏等任何互动功能时，自动跳转登录界面。


（3）页面功能：实现三种方式的登录——微信登录、手机验证码登录、QQ登录。
（4）页面逻辑内容及交互详细说明
① 微信登录
初始页面，软件logo下呈现微信登录方式，点击微信登录按钮进入微信平台授权登录界
面。授权完成后，完成登录，转入首页页面。
②QQ 登录
初始页面下，正下方有其他方式登录按钮，点击呈现其他方式登录提示词，点击QQ登录
进入授权页面。授权完成后，完成登录，转入首页页面。
③ 手机号登录
初始页面下，正下方有其他方式登录按钮，点击呈现其他方式登录提示词。 点击手机号
登录进入授权页面。 授权页面下呈现手机号码输入栏、验证码输入栏、发送验证码按钮
。 手机号码输入限制11位数字，当输入非数字内容时，输入界面不显示任何内容；输入
数字小于11位或大于11位时，发送验证码为灰色字体，无法点击发送。 点击获取验证
码后系统自动发验证码短信给输入的手机号，验证码按钮变为60s倒计时，倒计时结束前
不可再获取验证码。 验证码正确完成登录，转入首页页面。

五、页面详细功能说明
5.1 首页——热门页面
（1）页面名称：首页——热门页面
（2）页面入口：启动拼多多app直接进入“首页-热门“页面，或点击主菜单”首页“
按钮进入
（3）页面分布：
搜索区域； 商品分类区域； 活动Banner栏； 活动icon区域； 百亿补贴区域； 商品列
表区域。
（4）页面逻辑内容及其交互详细说明


1）搜索区域
位于首页页面最顶端； 点击搜索输入框页面跳转，底部自动弹出键盘，商品输入后点击
顶部右端”搜索“按钮进行搜索。也可以点击搜索输入框最右端的相机icon拍照或选择
照片搜索； 搜索页面提供”最近搜索“、”更多搜索方式“、”搜索发现“等功能； “
最近搜索”提供最近的搜索记录，右端有删除按钮，点击可以删除搜索记录。”更多搜索
方式”有搜索店铺和拍照搜同款功能。“搜索发现”下面智能推荐一些商品，点击商品跳
转对应的商品详情页。
2）商品分类区域
位于搜索框下端，分为热门、手机、内衣、百货、电器等； 左右滑动（除活动Banner栏
和活动icon区域以外的手机页面）进行切换，或者点击标题按钮进入对应页面。
3）活动Banner栏
位于页面中上部区域； 共9个Banner栏，每隔3秒自动切换，或滑动Banner栏区域
进行手动切换； Banner栏切换时其右下角对应的圆点变为红色，其余圆点为白色，点击
Banner区域页面跳转。
4）活动icon区域
位于页面中部； 不同的icon对应不同的入口，点击icon页面跳转，左右滑动icon区域
进行切换，切换时下方的红色矩形块进行相应的移动。
（5）百亿补贴区域
位于首页页面中部； “百亿补贴”下方展示商品图片，图片的左下角显示商品品牌，商
品下方显示“补贴价xxx元”； 此栏右上方显示标题全部补贴品，点击“全部补贴品”
或者任一商品图片跳转到同一个页面。
（6）商品列表区域
位于首页页面下方，上下滑动浏览更多商品列表； 区域左边展示商品图片，右边展示商
品详情，从上到下依次是商品名称、商品目前情况、商品价格、拼单数、同款商品排行榜

等； 点击商品列表区域，进入对应的商品详情页，详情页展示商品的的详细信息，可上
下滑动查看。

六、总结
通过对拼多多产品需求文档的撰写，有以下看法：
（1）平台内容化，打开拼多多进入首页可以看到多多果园、多多赚大钱、多多爱消除等
一些内容化的应用，在个人中心也安排了内容入口。这说明拼多多除了用户裂变拉新也在
致力于用户的留存，建立流量池，挖掘用户最大的价值。
当用户规模和增量不在，增加存量用户的价值就能使平台的价值最大化，目前拼多尝试通
过游戏和内容的入口，提高用户使用时长、促进用户黏性，触动用户隐形消费（隐藏的需
求被挖掘出来）和冲动消费（非必需消费增加）等。
但是其首页内容入口和活动入口没有明确划分，没有过多引导用户，应当考虑在无目的购
物和目的模糊下如何引导用户进入内容入口，同时是否应该根据用户画像进行区分，以此
提高用户活跃度。
（2）直播模式，比如直播首秀只要三位好友组团就能获得直播商品的五折优惠券，对用
户裂变的扶持力度非常大。但是拼多多直播模式相比淘宝、京东并不火热，现在拼多多也
没有直接的直播入口，唯一入口放在了”关注“界面。
目前市场上头部带货网红高度集中化，贡献了绝大数流量，另一方面头部网红能产生带动
效应来带动整体发展，比如李佳琦、薇娅火热后淘宝主播的人数增长明显加快。拼多多缺
乏网红孵化机制，走自我孵化的道路也十分艰难，可以考虑平台空降主播或者将用户看直
播买东西融入到购物场景、流程当中，同时精准直播，主播分类带货，卖美妆的只做美妆
，卖鞋的只卖鞋。
拼多多作为当下十分热门的电商app，其功能及信息架构十分复杂，涉及交互众多。作为
一个菜鸟，希望通过倒推拼多多的产品需求文档来提高对流程图、prd的认识，锻炼文档
撰写的能力，还请前辈们多多批评指正。
