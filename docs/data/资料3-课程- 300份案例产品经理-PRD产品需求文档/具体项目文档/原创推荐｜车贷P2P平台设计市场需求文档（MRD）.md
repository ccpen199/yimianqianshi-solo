（MRD） 
哈罗~大家好， I’m anna~~ 前不久上传了一个商业需求文档 (原创推荐  | 互金商业需求文
档BRD)，作为产品狗的三大文档， BRD/MRD/PRD, 还有剩下 2类文档没有给出来，想着抽个
时间写成一个系列，把产品狗的三样神器凑凑起吧 ~ 
本篇MRD继续采用 PPT的方式展示出来，在大部分实际操作中，很多创业型企业或者规模
比较小的企业会跳过 BRD和MRD，直接从原型开始，但是如果能在空余的时间里把 MRD自
己尝试写写，会对产品有更宏观的理解，对以后的产品规划其实是帮助非常大的，本篇
MRD是针对一个车贷 P2P产品设计的，主要从市场、用户、产品三个宏观的层面分析，是
对后面更为细化的 PRD的一个基础前提，虽然这与之前上传的 BRD并不属于同一个产品，
但是其实写作的思路是贯穿和相通的，放上来也希望得到大家的建议，一同学习，共同进
步！
