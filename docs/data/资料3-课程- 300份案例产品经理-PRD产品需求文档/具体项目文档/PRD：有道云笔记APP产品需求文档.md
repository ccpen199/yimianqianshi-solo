说明笔记记录已经成为生活/工作中的一部分。随着知识工作者数量的增加，国内效率办
公应用的需求会不断扩大。
尽管近年来出现了越来越多笔记产品，但由于切入时间较早，拥有一定的用户基数且不断
提供产品更新与服务的有道云笔记已经成为这一市场的领头羊，并且活跃人数也较为可观

其中“优化更新”、“同步问题”、“系统卡顿”、“数据丢失”、“系统问题”五个问

问题一：优化更新
许多用户希望有道云笔记可以：
增加使用空间； 可以自行选择图片压缩大小； 新增文件导出功能； 新增思维导图； 可以
对笔记文档进行批量管理； 新增夜间模式； 新增笔记内查找内容； 优化语音转文字功能
。
问题二：同步问题
部分用户在反馈中提到手机端与PC端不同步、内容生成后同步失败的问题，云同步是有
道云笔记的核心竞争力，如果该功能无法给予用户良好的体验，则难以吸引用户继续使用
有道云笔记，并容易流失忠实用户。
问题三：系统卡顿
部分用户在反馈中提到软件使用时会出现延迟、打开慢、反应慢等系统卡顿问题，使用户
不能及时进行笔记，降低用户体验。
问题四：数据丢失
部分用户在反馈中提到更换设备后笔记消失、笔记刚完成就丢失以及更新后之前的文档笔
记丢失的情况，这使得用户容易对有道云笔记失去信心，转而使用有道云笔记的其他同类
竞品。
问题五：系统问题
部分用户在反馈中提到在使用时有道云笔记会出现bug，导致无法进行正常使用。
问题六：其他问题
非会员广告过多； 产品迭代后，无法搜索笔记内容； 缺乏客服这样的问题咨询、帮助及
反馈的渠道。

七、总结
有道云笔记APP定位于商务办公领域下的笔记类产品，页面较为简洁，整体色调较一致
。而随着产品的不断更新迭代与功能的不断得到拓展，有道云笔记也获得了越来越多的高

收入、高学历人士的青睐（2018年有道云笔记发布的《80、90后高薪人群办公习惯洞察
报告》显示其用户72.97%月薪过万，96.83%的用户拥有本科及以上学历）。
而未来有道云笔记的发展应该不忘初心，牢记使命，紧紧把握住“云”+“笔记”这两个
核心点，这是笔者认为的用户的关键性需求，将此做到极致，会使得有道云笔记可以牢牢
把握住用户。而在满足了以上两个核心点后，有道云笔记可以增加其他相关的便利功能，
提升用户满意度。根据上文的需求池以及笔者对此产品的思考，提出以下一些产品优化建
议：
7.1 “云”体验【云同步（解决数据丢失）、云协作】
保证系统的稳定性，同时及时提醒用户的网络状态、其他设备对同一文档的作业情况与作
业进度以及笔记、文档的同步状态，确保云同步的成功以及解决用户数据丢失的问题； 
用户分享的笔记、文档可以设置相应权限，使得用户可以进行多人实时在线的云端文档协
作。
7.2 编辑体验（增加笔记模板或开放自定义模板、编辑功能、素材库、任务日历清单、脑
图）
新增更多官方的实用笔记模板，以及设立用户自制模板广场，使用户可以在有道云笔记上
制作自定义模板并进行分享与下载其他自制模板，满足用户对于笔记使用的多样需求； 
编辑中提供的功能可以参考word，为用户提供更加细腻的功能选择，比如用户可以调整
笔记内行距、在无序列表可以选择自定义符号、有序列表可以自定义样式等；同时可以在
编辑功能中添加批注功能，使用户可以对笔记添加批注；并且也可以增加查找与替换功能
，满足用户需要将一些字节进行替换的需求； 设立素材库功能，解决用户对于某一经常
需要添加的元件但是又要重复寻找元件进行添加的困扰； 设立任务日历清单，通过获取
用户日历的权限，使用户可以使用任务日历清单编辑当月工作任务、计划； 增加脑图功
能，使用户可以在有道云笔记中新建脑图笔记，并且完善相关脑图的功能细节。
7.3 视图体验（个性需求（主题、夜间模式））
可以推出多款不同的主题、皮肤，满足用户对于视图体验的需求，同时主题或皮肤可以推
出免费款、会员款、独立付费款，这也将成为推动用户付费的一个增长点； 推出夜间模
式，满足用户在不同使用场景对于视图的需求。

7.4 文档管理体验（批量管理文档笔记、一键导出笔记文档）
使用户可以对账户所有的笔记、文档进行批量管理，而不是只有文件夹内的笔记可以被用
户进行批量管理； 使用户可以将自己的笔记文档进行一键导出，并可以选择导出的格式
。
7.5 其他体验（客服反馈、使用引导、一键演示笔记文档、内容搜索）
提供客服反馈渠道，满足用户对于问题咨询、帮助及反馈的需求； 添加使用引导或新手
指南，使得用户可以快速上手软件及了解新功能的使用，避免因产品迭代造成顾客满意度
下降； 在笔记中添加一键演示笔记文档功能，满足用户对于笔记、文档自动演示的需要
； 优化内容搜索功能，使搜索功能从现在的仅能检索到笔记文档的标题，到可以检索到
笔记文档的内容，帮助用户更好更快地检索到目标笔记文档。
