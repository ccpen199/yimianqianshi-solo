二、市场及行业分析
2.1 传统室内设计行业
传统室内设计很大程度上依附于房地产开发行业，伴随着房地产开发企业的倒闭、转行，
装修设计公司也正在一茬一茬地倒掉。不少装修设计公司都面临类似的处境——一些小型
设计工作室由于没有项目不得不停业，大中型的建筑设计公司纷纷采取裁员降薪的方式，
压缩人力成本。
不仅经受业务市场容量减少的压力，传统室内设计行业还承受互联网的猛烈冲击，许多室
内设计师在客户难求的高压中度过。传统室内设计行业正处在变革期，设计师将会从房地
产开发流向再装修或与从事与其他产业相结合的工作，尤其是互联网。
而互联网的并发性与不受地域限制的特点，也会为设计师提供更多客户资源，更多元的盈
利方式，为用户提供直接方便的设计资源。
2.2 互联网+装修
齐家网、最美装修、蜗牛装修等互联网装修巨头，将传统家居装修设计服务扩大到互联网
领域，使得家居装修更加方便快捷，提供一个设计师与用户的交流平台。但这些装修巨头
为设计师提供的仍是传统的服务模式，即设计师需完成与客户接头、交流需求、定制设计
、装修改造整个过程，才能产生盈利。
这种服务模式成本持重，因而互联网装修市场目前仍局限在大户型家装。而又因服务人群
较为狭窄，通过同一用户反复盈利难以实现，互联网装修公司很难通过用户量再扩大进一
步降低运营成本，因而互联网家居装修设计仍为一个高消费的服务行业。
房价水平的不断上涨，使得微、小户型尤其是公寓和出租屋的市场规模进一步扩大。加之
人们对生活品质的追求越发热切，微、小户型装修设计市场潜力巨大。因而互联网装修设

计行业想要进一步发展，必须打破市场壁垒，进军微、小户型市场。而要进军微、小户型
市场，必须降低装修设计成本，即必须改变单次不可持续的服务模式。
2.3 电子商务行业
电子商务经历三个发展阶段。
电商1.0：即以早期电商eBay以及后来在国内发展起来的淘宝，京东等为代表的平台电
商。这类型的电商商品的发现机制是“检索”加“搜索”，这类电商构建商品交易平台，
允许众多商户入驻，商品种类虽全，但繁多复杂，缺乏商品筛选整理机制，用户决策成本
相对偏高。 电商2.0：即以过去6-7年兴起一批垂直电商为代表，包括聚美优品、兰亭集
势、以及目前规模最大的唯品会等。此类电商专注于某一类商品，类似于专卖店。 电商
3.0：即以近1-2年成长起来的一批具有“移动+社交”特性的电商。这种电商类似于买手
店，与移动电商的个性化、场景化的定位类似，其主要服务于对某种定位或调性的商品有
兴趣的消费者。通过各种方式进行个性化的推荐与发现，维护自己的客户群体。相比电商
2.0，这种模式更小更美。因而具有电商3.0特质的垂直电商网站和APP近两年市场占领
速度迅猛。
因此，通过电商的发展情况可推知，未来的电商发展将基于电商3.0，通过更加精准推荐
和定位，营造消费文化，为用户提供更为直接高效的购物体验。
2.4 青年人微、小户型软装家居消费市场
随着生活条件的提高，人们对生活品质及个性化的追求越发强烈，尤其是张扬个性、标榜
独特的当代年轻人。简陋、趋同一致、缺乏个性的居住环境早已不能满足青年人的需求。
但符合青年人居住微、小户型软装饰市场尚未成规模，也没有一条完整的市场供应链，生
产商提供的产品样式少，种类不齐全，加上零散散布在各种经销商手中，因而年轻人采集
装饰材料，搜寻相应的家居用品花费时间成本过高，阻碍了市场的进一步扩大。
另外学生自身设计能力不强，装饰家居环境经常有心无力，效果有限，远远不能满足他们
的需求。
市面上暂未有上市企业公司进一步挖掘年轻人的微、小户型软装饰家居用品需求，提供全
面，简便的宿舍装饰方案，供给远小于需求，因而这是个巨大的潜在市场。

三、用户分析
3.1 目标用户
（1）目标用户需求分析
在当下互联网社会，越来越多的年轻人，对自身生活质量的要求不再局限于温饱，更多是
个性化的生活追求。 在对待居住空间的定位上，满足基本住房的同时，年轻人也渴望能
够提高生活品质，享受更好的享受生活。 年轻人的消费观和对于审美、品质、性价比的
需求与上一代有着极大的差别，他们需要找到符合新生活方式的商品。
（2） 目标用户群体
有软装饰需求的18-30岁青年人； 需要购置家居用品的18-30岁青年人； 对家居生活品
质有一定要求的18-30岁青年人。
（3）目标用户特征
20-30岁的年轻人的居住环境及装修需求大致有以下三种：
还没毕业还居住在大学宿舍中，需要临时改装宿舍，只能软装，改善住宿环境的预算较低
。 毕业刚出来实习工作，居住在相对简陋的出租房内。需要购买少部分家具，有软装需
求，但预算也较低。 工作几年，刚刚有钱买房，供起了户型较小的一房一厅，可以硬装
和软装相结合，需要设计方案，装修预算相对较高。
共同的特点：属于微、小户型，布局相对单一，相对简陋。有改善居住环境的需求和预算
，但预算较低。

4.2 竞品分析
（1） 产品战略定位
半糖APP
半糖APP的Slogan是“我的生活态度·发现值得买的好东西”，可以看出：半糖的定位
是在当前型电商平台的暴露出不可避免的弊端而应运而生的。
电商的出现颠覆了传统的商业形态，虚拟货架不受物理空间限制，海量商品必然会导致发
现和找到商品的效率底下，大量的不感兴趣的商品的曝光伤害了用户逛和找的体验。
以淘宝、京东为例：这类大型电商存在一定的缺陷，商品种类繁多，且商品重复率过高，
导致用户无法便捷找到自己想要的产品，而这类平台里对商品的评价也存在过多的虚假信
息，对用户来说并不具备参考价值。
传统电商提供的只是信息对接的价值，随着电商设施的基础化越来越好，后端效率的提升
有限。而半糖APP则提供了在前端的信息对接上，实现更高效和体验更好的电商形态，
形成围绕人群的超级入口+供应商的模式。


半糖提升发现体验，通过用户的兴趣数据进行商品分发，精准的推荐简化了商品搜索和浏
览的流程，为用户提供精美好物。而且通过“找劵”和“券后价”可以为用户省下不少的
钱。
目前半糖的优势在于：为用户提供精致好物的同时为用户省时省钱。它作为一个供应商入
口类的应用，更多的偏重仍在与电商内容，社交和UGC的属性偏弱，但由于它是小众的
入口类电商APP，更多的其他属性可能会增加其运营成本。在这个层面上，启发我们初期
应做好产品核心功能的架构和深化。
最美装修APP
最美装修作为齐家App的一个补充，主要以内容为卖点，建立家装社区，增加用户粘性
和留存。
最美装修的存在：
一是为齐家中家装O2O业务引流； 二是通过偏内容的导购，扩充了齐家的业务线，扩充
家居用品、家电等泛家装业务，扩充了原本单一的家装业务。
从2014年7月至今已经迭代数十次，建立了较为完整的内容社区。最美装修借助齐家汇
集了大量的精英设计师，拥有大量的用户基础和技术庞大的精美原创家居设计实景案列，
还有大量真实的用户装修日记作为参考。
产品导购和提供免费报价和免费监理，为最美装修圈住了大领导粉丝。它的产品结构较为
完备，内容是产品的主要功能，引流和O2O业务才是该产品盈利的核心所在。
该产品主要目标是针对大中户型的互联网+装修，与我们陋室APP针对小户型家居装修业
务市场不相冲突。但其目前已经拥有的大量的具有高消费能力的忠实用户，和质量较高的
设计师资源，和市场认可度和品牌知名度。一旦进军微、小户型家居装修市场，将会对陋
室APP造成重大威胁。

最美装修APP：一个为齐家引流并主打家居装修领域的社区产品。利用较强社区属性和社

4.3 竞品优缺点分析
（1）半糖APP优缺点分
优点：
利用精品的挑选和推荐，同时通过折扣额优惠券，可以吸引大量关注性价比的用户。 精
品好物的挑选+较成功PGC运营，为用户提供优质的产品，相对于商品较多且信息繁多的
传统电商，用节省了用户搜寻商品的成本。 专注打造传统电商平台的优质超级入口，电
商属性较强，盈利模式清晰。
缺点：
通过折扣额优惠券，可以吸引大量关注性价比的用户的同时，流失掉了大部分关注高档产
品的用户。 电商属性较强，社交和社区属性可以忽略不计，容易造成用户的粘性不足。 

差异化不足，不管是与淘宝、天猫之类的电商平台相比，还是与小红书、网易严选等平台
相比，半糖APP仍然没建立起足够高的进入壁垒。
（2）最美装修APP优缺点分析
优点：
社区和社交属性较强，UGC和PGC优质内容的存量都较高。用户的粘性、留存和转化效
率较高。 产品从装修报价预估到装修售后服务，形成了完整的用户消费的闭环。 产品UI
较有设计感，符合产品的调性和风格，用户体验较好。 有齐家网站流量的引流、设计师
团队以及市场品牌作为基础，未来的发展潜力很大。
缺点：
作为一个较为成熟的产品，尚未建成自己的电商体系，未能把握住淘宝等传统电商的流量
。 产品的结构层次复杂，部分重要的内容所在层次较深，用户对产品的体验欠佳。
4.4 总结
半糖APP代表了为大型电商平台引流类的新型引流产品，而最美装修APP代表了装修+
互联网行业的精品产品。把这两个产品作为陋室APP竞品，可以分别学习这两类产品的
优点和经验，同时关它们的缺点和不足，以便使陋室APP扬长避短，更好地完善产品体
验。
五、产品概述
5.1 产品简介
“陋室”是一款针对微、小户型的年轻人软装饰家居生活推荐和品质生活用品导购的场景
电商平台。利用微、小户型的相对单一性为突破口，利用建模技术，将居住空间完全仿真
，并配上各种优选的商品和与装饰摆件形成具有个性化的主题居住空间。
通过设计仿真的软装修效果图来构建真实可观的居住生活场景，并利用商品有机搭配来向
用户推荐该商品的使用场景，使商品在空间和心理上的与用户零距离，从而激发年轻人打
造个性空间的欲望。

通过整体效果来进一步引导用户去浏览、购买场景中的商品，同时对接淘宝等电商平台实
现流量导入，流量变现以及持续盈利。
另外，利用自媒体配合宣传，定期推送精品内容，实现app引流。后期引入VR技术，进
一步使用户产生一种沉浸式的购物体验。

目标市场：大学生宿舍软装修市场。适用于宿舍的各种品质、有创意、实用类或摆饰品市
场。
目标用户：
想要改善宿舍环境却找不到渠道和方法的学生（以大一大二学生为主）； 实习租房，想
改善生活环境的学生（大三大四为主）； 想要找适用于宿舍的各种品质、有创意、实用
类或摆饰品的所有学生。
产品愿景：致力于创造新的宿舍生活方式和体验，成为最大的宿舍软装修商品平台。
核心功能：

场景模块：打造场景的图文展示、搜索和筛选功能。 专题模块：品质单品推荐，宿舍攻
略 个人中心：注册登录等功能。
