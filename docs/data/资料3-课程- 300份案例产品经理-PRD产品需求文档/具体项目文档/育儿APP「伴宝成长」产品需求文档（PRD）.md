# 育儿APP「伴宝成长」产品需求文档（PRD）

辞职在家带娃期间，下载过多款手机育儿App，但是没有一款满足我的切身需求。闲来无
事，设计了这款「伴宝成长」，分享出来和大家一起交流。这篇是产品需求文档，关于产
品背景，产品介绍，竞品分析，本文简略带过(若有较多人对这款产品感兴趣，后期将这
部分内容放到网上交流）。
「伴宝成长」是一款以书籍形式记载宝宝成长，分享宝宝成长和交流育儿心得的手机APP
。

1. 产品概括
1.1 背景
随着生活水平，人们文化水平的提高和科学育儿知识的普及，家长们对于留存住宝宝成长
点滴，多样化喂养和科学育儿的需求越来越浓烈。不再满足单一用手机或相机将宝宝照片
拍摄留存的方式，更喜欢将宝宝成长的点滴在社交类平台进行分享，与此同时分享育儿中
的心得，寻求育儿困惑的解答。同时，家长们也更加注重宝宝饮食健康和种类丰富，穿衣
打扮的精致，亲子情感的提升等等。
市面上育儿产品数量较多，但是侧重点不同，某些侧重照片云存储，某些侧重育儿交流，
无法满足用户的多方面需求。
1.2 产品介绍
伴宝成长App是一款以书籍形式记载宝宝成长，交流育儿心得的产品。帮助用户记录宝
宝成长的时刻，抒发育儿过程中的苦闷，解决带娃期间的困扰，便捷购买宝宝的用品。让
宝宝成长的片段不流逝在时光中，科学养娃轻松带娃成为育儿的主旋律。伴宝成长包含五
大模块，成长书，交流，服务，商城和我。用户可独自或邀请家人在成长书中记录宝宝成
长的时刻。在交流中分享带娃心得，沟通育儿困惑。在服务中寻求线上和线下的帮助，轻
松带娃。在商城中优选宝宝的穿着，洗护，喂养，玩乐的商品，足不出户解决宝宝一切所


需。在我中进行帖子，咨询，订单等管理。通过伴宝成长，让带娃不在迷茫，困惑和无助
。
1.3 产品定位
让带娃变得轻松愉快，和宝宝的记忆拥有温度
1.4 目标用户群
有诗人情怀的家庭
