# 领克全球培训系统内训需求文档

这是一份关于领克全球培训系统内训需求文档，一起来看看~


一、概述
1.1 项目背景
LYNK&CO（领克）是欧洲技术、欧洲设计、全球制造、全球销售的新时代高端品牌， 具
有与生俱来的全球化基因，为全球都市人群打造，它诞生于互联网时代，将互联网思维与
传统汽车工业相融合，实现出行方式向个性、开放、互联的趋势变革。
1.2 目的和范围
为经销商人员“量身定制”的专门培训，课程以经销商人员为主要培训对象，可以以专业
销售课程，企业文化，精神面貌等各个方面进行培训。 线上组织经销商人员内训课程学
习和进行管理，并可对内训训课程进行审核，内训任务记录管理。 使受训人员的知识、
技能、工作方法、工作态度以及工作的价值观得到改善和提高，从而发挥出最大的潜力提
高个人和组织的业绩，推动经销商和个人的不断进步，实现经销商和个人的双重发展。 
本文档主要读者范围为产品研发部的产品经理、前后端工程师、UI设计师产品运营专员
、测试工程师。
1.3 产品概要
主要是主机厂对经销商，经销商店内内训任务管理的一个操作平台，主机厂建立A类内训
任务下发给经销商，经销商针对主机厂内训任务进行内训课程计划安排，上传课程文件和
图片、视频等，主机厂并对经销商内训任务进行审核。 经销商管理员自主建立B类内训
任务，添加内训课程计划，指定店内人员参加学习，并对其进行内训审核。
1.4 产品定位
建立内训任务，主机厂和经销商进行内训任务审核的一个操作管理平台。

二、内训任务列表
用户场景：主机厂查看内训任务情况。 功能描述：内训任务列表菜单按钮。 输入/前置条
件：点击内训管理下拉列表。 需求描述：内训任务列表菜单。 输出/后置条件：点击【内
训任务列表】按钮进入内训任务列表页面。


2.1 查询
用户场景：主机厂查看内训任务。 功能描述：内训任务查询，显示内训任务列表。 输入/
前置条件：输入框内输入任务内容。 需求描述：进行内训任务查询，查看任务名称、开
始结束时间、内训、考核方式、学时时长、状态。 输出/后置条件：点击内训列表【编辑
】可对内训任务进行修改编辑。


2.2 添加A类内训任务
用户场景：主机厂建立添加内训任务。
功能描述：通过文本选项建立A类型内容，对其添加发布。
输入/前置条件：点击内训任务列表【+添加A类内训任务】按钮进入内训系统管理A类
内训任务面。
需求描述：
填写相关内训名称、起止时间、内训范围。 培训和考试方式选择并勾选项，并设置时长
。 填写内训任务内容，点击【+选择文件】通过本地路径选择文件，点击【开始上传】完 
成文件上传。
输出/后置条件：
点击【保存】按钮完成建立A类内训任务。 点击【发布】按钮所建内训任务下发给经销
商，且发布仅可发布一次，不可撤回。


三、内训执行列表
用户场景：经销商查看内训执行任务情况。 功能描述：内训执行列表菜单按钮。 输入/前
置条件：点击内训管理下拉列表。 需求描述：内训执行列表菜单。 输出/后置条件：点击
【内训执行列表】按钮进入内训执行列表页面。


3.1 添加B类内训任务


用户场景：经销商建立添加B类内训任务。
功能描述：通过文本选项建立B类型内容，对其添加发布。
输入/前置条件：点击内训执行列表【+添加B类内训任务】按钮进入内训系统管理B类
内训任务面。
需求描述：
输入框填写相关内训名称、起止时间、内训范围。 培训和考试方式选择并勾选项，并设
置时长。 输入框填写内训任务内容，经销商在内训任务学习期间完成资料上传工作，点
击【+选 择文件】通过本地路径选择文件，点击【开始上传】完成文件上传。 点击【保
存】按钮完成建立B类内训任务。
输出/后置条件：点击【保存发布】按钮提交给主机厂待审核。
3.2 操作
（1）课程计划
用户场景：经销商编辑添加内训课程计划。 功能描述：查询内训课程计划。 输入/前置条
件：点击内训列表【课程计划】按钮进入到该页面。 需求描述：输入关键字和课程名称
查询内训课程计划,生成内训课程计划安排列表。 输出/后置条件：点击【内训列表】按钮
返回到内训列表页。
（2）添加课程安排


用户场景：经销商自主编辑添加课程。
功能描述：课程安排计划建立。
输入/前置条件：点击内训课程安排安排【+添加课程】按钮进入到课程安装页面。
需求描述：
显示课程安排名称和培训开始结束时间。 根据输入框标题填写课程名称、老师名称、勾
选日历设置开始结束时间以及课程内容。
输出/后置条件：点击【保存】完成添加课程操作，返回内训课程安排列表页面。
（3）报名设置


用户场景：经销商建立内训课程计划和报名。
功能描述：内训任务课程报名。
输入/前置条件：点击内训列表【报名设置】按钮进入到报名页面。


需求描述：
填写相关内训名称、起止时间。 培训和考试方式选择并勾选项，并设置时长。 填写内训
任务内容。 通过内训人员报名列表进行勾选参加内训人员。
输出/后置条件：点击【报名】完成内训报名操作，返回内训安排列表页面。
（4）上传提交
用户场景：经销商编辑上传内训执行。
功能描述：建立编辑上传内训执行课程。。
输入/前置条件：点击内训执行列表【上传提交】按钮进入到内训执行页面。
需求描述：


填写相关内训名称、起止时间。 培训和考试方式选择并勾选项，并设置时长。 填写内训
任务内容，在内训任务学习期间完成资料上传工作，点击【+选择文件】通过本地路径选
择文件和视频（大小30M以内），点击【开始上传】完成文件上传。
输出/后置条件：点击【完成内训执行】按钮返回到内训执行列表。
四、内训执行查看


用户场景：主机厂或者经销商查看内训执行情况。 功能描述：内训执行查看菜单功能按
钮。 输入/前置条件：点击后台系统首页内训管理菜单。 需求描述：内训审核管理菜单。 
输出/后置条件：点击【内训执行查看】按钮进入到内训查看列表页面。
4.1 查询
用户场景：主机厂查看进行经销商内训列表。
功能描述：显示经销商内训执行列表。
输入/前置条件：点击菜单列表【内训执行查看】按钮进入到该页面内容。


需求描述：
通过输入经销商名称和经销商代码，勾选审核状态查询。 显示区域、经销商名称、代码
、内训任务名称、计划、执行课时、要求课时、开始结束时间、状态是否提交情况，点击
【查看】进入详情页面。
输出/后置条件：点击【查询】下列生成经销商内训列表。
4.2 操作
查看：
用户场景：主机厂查看审核内训详情。
功能描述：内训主题，开始结束时间。
输入/前置条件：点击内训执行查看列表【查看】按钮进入到审核内训页面。
需求描述：
内训主题、开始结束时间和内训课程计划列表。 内训课程计划列表，可添加内训照片、
视频和其他附件内容。
输出/后置条件：点击表头【内训审核管理】按钮返回。


五、内训审核管理
用户场景：主机厂查看内训审核情况。 功能描述：内训审核管理菜单按钮。 输入/前置条
件：点击内训管理下拉列表。 需求描述：内训审核管理菜单。 输出/后置条件：点击【内
训审核管理】按钮进入内训审核管理页面。


5.1 查询
用户场景：主机厂对经销商内训任务进行审核。
功能描述：输入框内输入任务名称查询。
输入/前置条件：输入框内输入任务内容。
需求描述：
输入关键字和内训审核对其查询。 显示内训任务名称、类型、开始结束时间、内训考核
方式、计划和完成率，可以对其内训任务进行查看和导出操作。
输出/后置条件：
点击【查看】进入到经销商内训情况列表。 点击【导出】生成文件保存到本地。
5.2 操作
查看：


用户场景：主机厂查看经销商内训情况。
功能描述：显示经销商内训情况。
输入/前置条件：点击内训任务列表【查看】按钮进入到该页面。
需求描述：
选择状态下拉项，按类别选项查询。 显示大区、经销商名称、代码、内训名称、计划、
执行计划、要求课时、审核状态经销商内训情况列表。
输出/后置条件：点击【内训管理】返回到首页。
补充说明：点击内训任务列表【导出】生成文件保存到本地。


六、经销商内训审核
用户场景：经销商查看内训审核情况。 功能描述：经销商内训审核列表菜单功能按钮。 
输入/前置条件：点击内训管理下拉列表。 需求描述：经销商内训管理审核菜单。 输出/
后置条件：点击列表【经销商内训审核】可进入到经销商审核页面。


6.1 查询
用户场景：主机厂对于经销商内训任务区域查询审核。
功能描述：通过勾选项表头和搜索得到经销商内训审核列表。
输入/前置条件：点击内训管理下拉列表【经销商内训审核】按钮进入到该页面。
需求描述：
大区、战区、巡回员标签下拉选项，输入关键字段或者经销商名称/代码进行查询，点 击
【查询】按钮。 经销商内训列表点击【审核】按钮可对进行操作。
输出/后置条件：点击列表【审核】可进入到经销商审核页面。
6.2 操作
（1）审核


用户场景：经销商查看内训情况。
功能描述：显示经销商内训任务概况，显示内训名称、任务类型和开始结束时间，审核状
态是否提交。
输入/前置条件：点击经销商内训审核列表操作菜单【审核】按钮进入到该页面。
需求描述：列表表头显示大区、经销商名称、内训任务类型、开始 结束时间、课程计划
、执行课时、要求课时、提交时间和审核状态。
输出/后置条件：点击列表【查看】按钮进入到审核内训页面。
（2）查看


用户场景：主机厂查看经销商审核内训详情。
功能描述：内训主题，开始结束时间。
输入/前置条件：点击内训执行查看列表【查看】按钮进入到审核内训页面。
需求描述：
内训主题、开始结束时间和内训课程计划列表。 内训课程计划列表，可添加内训照片、
视频和其他附件内容。
输出/后置条件：点击表头【内训审核管理】按钮返回。
