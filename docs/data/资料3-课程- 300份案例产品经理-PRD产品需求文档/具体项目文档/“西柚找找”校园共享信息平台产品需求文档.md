# “西柚找找”校园共享信息平台产品需求文档

编辑导语：西柚找找是一个为学生、教职工、学校和校周边用户群体提供信息便利的一个
平台，它针对庞杂的校园信息进行汇总分类，用户通过学生证，身份证进行实名注册，根
据需求选择服务类型。本文作者对西柚找找进行了深度分析，并且制作了一份产品需求文
档，希望看后能够对大家有所帮助。

1.2.1 产品类型
互联网校园信息共享平台


1.2.3 产品介绍
西柚找找针对庞杂的校园信息进行汇总分类，用户通过学生证，身份证进行实名注册，根
据需求选择服务类型，是为学生，教职工，学校和校周边用户群体提供信息便利的一个平
台。
1.2.4 产品定位
西柚找找是一个以校园为中心实现生活、学习和交友的信息共享服务平台，包括二手交易
、失物招领、跑腿服务、兼职招聘、房屋租赁等各类信息，通过线上信息平台和线下服务
站的结合，实现信息的高效利用和流通的服务系统。


1.3 产品用户
西南石油大学在校大学生，教职工，学校及学校周边人员。
1.4 需求总结
根据KANO模型，将用户需求分为五类：基本型需求、期望型需求、兴奋型需求、无差
异型需求以及反向型需求。

六、总结
在大学校园生活里，极大量的需求信息喷涌而出.然而，校内信息共享平台众多，消息冗杂
，寻找需求信息需耗费一定的心力。
信息需求者与供应者总是失之交臂，彼此寻找，不能有效实现信息的流通与对接。
西柚找找服务系统的目的就是将这些庞杂的信息汇总起来，进行分类，优化平台服务，从
解决校园生活的各种需求出发，使信息能够高效的流通起来，真正为信息用户服务到位。
目前西柚找找校园共享信息服务系统主要存在以下几个创新点：
后台管理增加了学生群体，在减轻学校负担的同时，也为学生提供了福利，实现了双赢。 
增加了一部分特征学生群体的积极参与性，该部分学生可通过失物招领和献策等获得好人
分，可作为加综测或二课分的凭证，或者申请服务站点管理者，也可获得加综测权限。 
通过线上服务平台和线下硬件服务端的结合，加速了信息的准确定位和流通。 无效信息
处理：对于失物招领服务模块，超过60天无人认领物品的处理信息在公布栏中进行公开
。既将无人认领的物品的信息进行贴出，包括捡拾物的名称、捡拾地点、时间、捡拾人姓
名和联系方式，然后再公示出失物招领处的处理方式。比如贵重物品电脑，手机之类的，
60天无人认领就交给当地派出所，其他物品则选择捐献给希望工程。对于其他服务模块
，信息自动失效，由专人进行处理销毁。
