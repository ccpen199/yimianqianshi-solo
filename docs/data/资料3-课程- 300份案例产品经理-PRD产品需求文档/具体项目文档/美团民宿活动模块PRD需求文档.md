供个性化民宿住宿体验，为年轻旅行者外出旅游提供旅行住宿，其特点是房源个性化、多

1.1.2 市场分析
从2011年底国内出现第一家短租平台爱日租，紧接着途家、木鸟短租、小猪相继成立，
国内涌现出一批在线短租平台。
政策的支持、资本的加持、个人房东和房源数量的增长、综艺节目的普及、共享住宿的观
念深入人心等一切行业利好因素推动中国共享住宿行业蓬勃发展。
在经过2018年的发展热潮之后，2019年整个共享住宿市场发展趋于平稳，投资者回归
理性，各大平台也纷纷迈入7年之期，行业逐渐规范和成熟，行业等级也逐步分化，途家
民宿、木鸟民宿、美团民宿、小猪民宿、airbnb挤进国内民宿行业的第一梯队。
产品定位而言，airbnb和途家民宿走国际化路线，注重出境游等全球化市场，其中途家
民宿以商务旅行为主。


木鸟短租为以家庭为主的群体出行提供住宿空间，和品牌合作提供中高端品质房源。小猪
民宿走经济路线，注重下沉市场。美团民宿将目标锁定在了年轻用户群体上。总体而言，
各平台间各有侧重和垂直领域，但彼此业务也有重叠。
1.1.3 需求背景
1.1.3.1 美团民宿现在面临的困境
根据2019年中国旅游和名宿发展协会发表的数据来看，四大本土民宿预定平台中，美团
民宿是持有在线房源数量最少的，仅有72万套，与占据民宿市场份额超过40%的途家相
比足足少了158万套。
虽然airbnb并未公布在中国持有的房源数，但是其全球房源数目已经超过了700万套。
其次，国内现有在线共享住宿平台的房源重复度高，同质化严重，房东房客游走在平台之
间没有平台归属感。
另外，对于房东而言，民宿行业前期投入过大，资金门槛高。对于大部分用户来说，民宿
预定都是低频需求，短期内大幅度提升新房源和用户量不现实。
因此，美团民宿要想在巨头林立的民宿市场中杀出一条血路来，仅靠现有资源是不够的。
1.1.3.2 寻找突破点
笔者根据网络数据和信息整理发现大部分用户选择民宿而非其他住宿平台的原因有一下几
点：
性价比高，同等价位民宿比酒店更舒适，家庭出行更经济。 生活设施配备齐全，拥有独
立宽阔的活动区域，可携带宠物。 干净、安全、服务好。 设计风格多样化，最赶潮流，
极具艺术性。 充满人文色彩，深入当地，与当地民俗文化结合。 有温度、个性化、人情
味浓，让人觉得温馨。
将竞争的重点转移到了体验和服务的第二战场，利用美团的平台优势，展开全产业链深化
发展路子的探索，覆盖食行游娱购等多文化领域的服务增值，似乎是可行之举。

1.5 术语与缩写解释
共享经济：一般是指以获得一定报酬为主要目的，基于陌生人且存在物品使用权暂时转移
的一种新的经济模式。其本质是整合线下的闲散物品、劳动力、教育医疗资源。
共享住宿：共享经济发展新出现的交易模式。共享住宿是依托互联网诞生的一个新业态，
近年来发展迅速，创新日益活跃，发展潜力巨大，在制定标准时要遵从审慎性、前瞻性、
可操作性原则，发展与规范同步。

5. 功能用例和页面详细说明
5.1 查找活动
5.1.1 功能介绍
用户按照活动名称、类型或关键词搜索查找活动，或增加条件筛选或排序活动并显示。
5.1.2 优先级
高（P0）

系统已存在正在发布且开展的活动。
5.1.4 后置条件
无

页面逻辑


用户点击导航中的“活动”，进入活动模块 用户添加想要开展活动的城市 用户按照需求
添加条件筛选查找或排序活动 系统根据用户添加的条件匹配显示活动
5.1.6 分支过程
用户按照需求添加条件筛选查找或排序活动 用户在文本框内输入活动名称、类型或关键
词搜索查找活动 利用页面选项增加条件，筛选或排序活动
5.1.7 异常情况
无满足条件活动：弹出toast弹窗，显示“抱歉，没有搜索到符合条件的活动哦”持续 3 
秒。 长按时与服务器失去连接：弹出 toast提示“网络好像开小差了，请检查一下你的网
络哦~”持续 3 秒。 在活动页面搜索房源信息：跳转到查找房源页面。
5.1.8 页面内容及交互说明


5.1.8.1 返回


点击跳转到首页页面
5.1.8.2 选择开展活动的城市
显示选择城市弹窗，选择完城市，在页面显示活动的模块将按照“当地热门”、“文化体
验”、“家庭活动”、“热门线上”、“附近活动”、“猜你喜欢”、“免费公益”进行
分区显示。
5.1.8.3 搜索框
可搜索活动名称、类型、活动关键词，点击提示内容消失，显示键盘，输入完毕点击搜索
按钮，活动显示模块显示包含搜索关键词的活动。
5.1.8.4 选择活动类型
分为6大搜索分类模块，分别是：“热门活动”、“体验当地”、“艺术之旅”、“限定
活动”、“Get新技能”、“个性化”。
“体验当地”模块包含“向导带领及制定旅行计划”、“体验或制作当地美食”、“摄影
跟拍”、“传统服饰租赁”、“体验当地传统活动”、“城市历史、名人文化追寻”、“
自然风光相关”、“另类探索城市”、“探索当地小众好去处”。
“另类探索城市”如探索城市收集印章等完成任务类型的活动，或者在旅行过程中设计故
事如寻宝，让用户参与其中，将用旅行变得更有意思。
点击“类型”，显示活动类型弹窗，选择弹窗左侧按钮，在右侧显示不同内容，类型选择
完成活动模块显示符合条件的活动。
5.1.8.5 选择开展活动的时间


点击“时间”显示时间弹窗，弹窗内显示近6个月日历。时间选择完成，活动显示模块显
示此时间段内正在开展且参与人数未满的活动。
5.1.8.6 活动排序
点击显示排序弹窗，排序规则分别是“离我最近”、“最受欢迎”、“好评优先”、“低
价优先”、“高价优先”。点击不同的排列规则，活动显示模块将按照需求显示排序后的
活动。
5.1.8.7 增加筛选条件
点击进入筛选页面，筛选条件包括“价格”、“活动时间段”、“适合人群”、“活动量
”、“活动量”、“是否需要提前准备物品”、“语言”。提交筛选条件后页面将跳转回
活动查找页面，活动显示模块将筛选后的活动显示出来。
5.1.8.8 显示活动模块
未选择城市地点前，按照活动在平台受欢迎程度分区显示“热门活动”、“文化体验”、
“热门线上”、“家庭活动”、“猜你喜欢”、“免费公益”。
选择地点后，按照“当地热门”、“文化体验”、“家庭活动”、“热门线上”、“附近
活动”、“猜你喜欢”、“免费公益”分区显示活动。
添加搜索内容、筛选条件和排序规则后，按照条件显示活动，不再分区。
5.1.8.9 收藏活动
点击活动将被收藏，可在“我的收藏”内查看。
5.1.8.10 开展中的活动
包含信息为活动封面、活动名称、活动类型、价格、评分、组织者头像和名字。
点击跳转该活动详情页。

5.4.5 主干流程

藏”按钮取消收藏活动。 系统将取消收藏信息更新至数据库。 用户查看收藏的活动。 系
统按倒叙将用户收藏活动显示到“活动收藏”页面中。

无
5.4.7 异常情况
