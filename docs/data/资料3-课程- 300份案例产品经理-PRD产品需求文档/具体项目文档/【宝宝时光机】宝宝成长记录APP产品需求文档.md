# 【宝宝时光机】宝宝成长记录APP产品需求文档

宝贝时光机是一款以视频拍摄为主，全家参与记录宝宝成长的工具，同时可以和亲友分享
宝宝故事，在社区交流拍摄经验。
一、文档说明
“宝贝时光机 ”是一款家庭参与的记录宝宝成长的视频APP； 在做这个产品之前没有做
详细的竞品分析，先看市场上已有的产品会限制思维，本文档立足于用户的需求； 计划
下一步做详细的竞品分析，然后对产品进行改进； 做出MVP后，再回到访谈用户，根据
他们的反馈对细节进行打磨。
欢迎直接粗暴的批评指正。


二、产品介绍
1. 背景介绍
记录宝宝成长是妈妈、爸爸和祖辈共同的需求
随着人们文化水平和生活水平提高，家庭基础消费向精神文化消费延伸，家长们对于记录
宝宝成长点滴的需求越来越强烈，爸爸的参与度也越来越高； 计划生育独生子女进入主
力育儿年龄阶段，老人对孙辈的关注度也越来越高，并且当代老人也开始接受育儿新方式
； 从备孕到宝宝四五岁，用户群体的活跃时间有将近六七年。
视频记录极具优势
随着高水平摄影摄像功能的智能机的普及，用照片、视频记录宝宝的成长越来越普及，视
频在记录和表达上有显著优势。
妈妈们需要专门的平台分享宝宝
妈妈对宝宝的爱超越一切，她们有强烈的分享欲望，朋友圈晒娃就表现的非常明显； 但
是很多妈妈意识到朋友圈频繁晒娃并不合适，观众很容易厌倦，因此妈妈们需要专门的平
台分享宝宝的生活，从而获得更强烈的幸福感。
二胎开放，人口红利

五、结语
即将毕业，正在找产品方面的实习/正职。初次写PRD，不足之处甚多，欢迎大家批评指
正，继续改进；也非常欢迎各位前辈交流产品方面的经验、想法。
设计这款产品的想法源于身边的亲朋好友对宝宝成长的记录的需求未被满足，市面上有很
多类似产品，但是要么侧重育儿知识、育儿经验分享，要么记录成长的功能不甚理想。
这款APP和普通的视频APP相比，突出了照片整理功能，家庭共同参与，另外添加了宝
宝元素，比如：贴纸。和其他记录宝宝成长的APP相比，突出了视频功能和宝宝摄影经
验交流。
