1.2 产品介绍
Forest 是一个帮助用户暂时远离手机，专心于目前工作的 App。
每当用户希望有一段专心工作的时间，用户可以在 Forest 中种下一颗种籽。在接下来的
时间内，这颗种籽将会慢慢地成长为一棵大树。若是用户禁不起诱惑，在这段时间内离开
这个 App 去看微博、玩游戏，用户充满生机、可爱的小树将会枯萎而死。每一天，都会
有用户有一片森林种满了。用户今天所有茁壮的大树以及枯萎的残枝，每一棵树都代表了
用户曾经努力的时光。
1.3 产品总结
这是一款控制用户使用手机保持专注的生产力工具。 以番茄工作法为基准。 用好看的UI
设计和集体种树等规则的模式来激励用户。
二、全局说明
2.1 功能权限
功能权限分为 登录/未登录 两个状态：

种植、添加标签、云端同步、购买树种、种植真树、添加好友功能。

2.4 APP规则说明
2.4.1金币
种植获取金币：（使用道具翻倍，手机工作模式减半）
大树：4+1/五分钟+5/额外三十分钟 小树：1+1/五分钟
成就获得金币：按照具体情况设定。
金币消费：
购买树种：第一次为500金币，后每次增加100金币，1000封顶。 购买音乐：单次
500金币。 种植真树：单次2500金币，最多种植5次。 删除记录：单次60金币。
2.4.2种植
种植的最小时间为10min，最多120min，调节的最小时间单位为5min。 20min以下
种植灌木，25min以上种植大树。 大树有25min、60min、90min、120min四种形
态，种植不同时间获得的树种形态不同。举例：种植90min，则可以获得25min、


60min、90min三种形态，若在89min时中途退出，获得三棵枯树。其它情况以此类推
。 种植的开始的10s内可以选择退出而不产生枯树。 设定为“种植后继续计时”，添加
时间数据但不增加树的数量。

七、测试异常页面及突发状况
7.1 网络异常
在断网的情况下仍可以继续种植和查询数据，但是无法执行添加好友、多人种植、同步等
功能 断网点击购买道具：弹出窗口后显示“无法连接到iTunes Store”。
7.2 多设备登录
不同的设备登录一个账号可以同时种植，时间范围取两者叠加，重叠部分数据不计入排行
榜和用户数中。
7.3 ios 与Android 系统
ios 与Android 系统用户之间能够看到用户信息，但是头像无法同步。


八、高优先级优化点
1. 优化多设备登录体验，尤其是数据同步问题
2. 解决森林页搜索字符点击返回后字符取消的BUG
