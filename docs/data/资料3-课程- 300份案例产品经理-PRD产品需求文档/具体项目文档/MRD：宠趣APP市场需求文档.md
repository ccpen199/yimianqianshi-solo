二、 市场及行业分析
2.1 宠趣APP产品背景
随着生活品质的上升，宠物市场规模越来越大。但是在宠物用品上没有合适的购买体验分
享，宠物养育上缺乏有效的经验分享平台，宠友们缺乏一个交流的渠道，
宠趣app是一款让用户放心购买宠物用品的同时，还可以分享宠物相关心得经验的平台
。用户可以在app中选择宠物用品购买，在圈子中分享自己的点滴故事，在社交中结识
更多类似相同趣味的好友，通过圈子——让你了解关与宠物相关的一切。
2.2 宠物市场概述
宠物市场及现状
国内宠物食品的销售渠道已呈现出线下宠物用品店零售和线上购买并重的趋势，但经过多
年发展至今未出现独角兽平台或企业，说起来是由于宠物食品消费行为形成较晚，导致行
业一直处在冷关注状态，既无序，又混沌。
质量安全：目前市场鱼龙混杂，无法让消费者信服平台宠物用品质量。
养宠经验：随着消费者对于宠物喂养观念改变，但是缺乏相关经验，一头雾水，不知如何
正确喂养。
传统宠物电商平台APP现状
目前宠物用品平台上论坛与商城关联性较差，宠物用品使用心得无法通过达人评测与商城
商品进行关联，对于宠物主选择上会存在一定的顾虑。宠物主之间缺乏交互性对于平台用
户黏度上会大打折扣。
2.3 宠物市场特征
宠物消费追求品质

随着经济水平的提高，宠物越来越受重视，高端消费者很少会购买中低端产品，另一方面
中低端的消费者对于高端产品的向往，使其在日常使用产品之外，也逐渐对高端产品进行
尝试。
消费升级宠物产业迎来井喷期
随着养宠队伍人数的快速增长、科学喂养方式的普及、以及养宠观念和宠物消费观念的改
变，越来越多的宠物养主追求宠物食品的质量、营养、安全。
2.4 行业简述
行业格局：
目前宠物电商平台，行业格局并不明显，没有行业龙头，没有市场知名度，业内熟知的也
就是E宠商城和波奇宠物。
E宠商城、波奇宠物都有宠物电商功能，其中波奇宠物以社交论坛为主打，目前用户粘性
最大，而E宠商城，专做中高端宠物用品电商，也拥有固定的用户群体。
E宠商城作为最早成立的专做宠物用品的产品，但是在购物流程以及用户粘性上还是存在
一定的不可取之处。波奇宠物作为后起之秀，不仅拥有成熟的论坛体系，还有了一套成熟
的电商购物模式，增加了VIP黑卡折扣，不仅增加了盈利还可以增加用户的二次购买欲望
。


行业发展：
“社交+购物”不局限于单独的商城和社交圈，希望在选择商品中增加社交诱导，在社交
中引流商品购物；通过多种途径让宠主充分了解养育习惯，养育所需用品，逐渐培养宠主
分享的习惯，以PGC的内容为诱因，逐渐转向UGC内容。鼓励用户分享，生成内容，增
加用户的黏性。
小结
（1）目前正是进入市场的好时期，应快速布局；
（2）重视社交与购物板块，给用户更好的体验；
（3）商品评价中鼓励用户分享测评报告，让更多用户了解商品的质量，安全，从而能够
放心购买；
（4）养宠人群增加，养育经验缺乏，可以考虑开设专业人士解答板块；
（5）增加用户打赏机制（礼物打赏，礼物可以用来提现），让更多优质内容创造者有动
力分享。
三、用户说明
3.1 目标用户
80、90后占大部分，用户多为宠物养主； 有经济条件：在保证宠物日常用品的同时追求
更深层的合理搭配； 学习养育经验，寻找志同道合的宠友，分享宠物日常。

（1）Lucy 19岁 学生 宿舍
想养宠，但是没有条件，搜索相关宠的帖子视频，满足自己养宠的心理安慰。 学习养宠
经验，交流更多朋友。
（2）Liliy 26岁 白领 公寓

（3）Jam 31岁 经理 自有住宅
对于宠物养育希望得到更加科学的喂养，更多关注于品质与安全，希望获得可靠高端的用
品，浏览其他宠物情况。 购买高端养育用品，交流更多志同道合的朋友。
3.3 用使用场景
场景一、购买宠物用品，社交分享
小李是生活在一线城市的白领，住在公寓，因为在陌生的城市里太孤单所以最近购买了一
个宠物。小李由于第一次养育没有什么经验，又希望宠物过得很好，但是网上的相关知识
太过繁杂，养育的方式千奇百怪，推荐的商品又没有可靠的购买渠道，小李因此感到苦恼
。
后来小李了解了宠趣APP，在这里发帖询问养育经验，根据预算得之相对应的商品推荐，
在商城中选择对应的商品，在满足自己养育宠物的同时结交了同城的宠友，在这个陌生的
城市里多了几分温馨。
场景二、浏览宠物帖子，满足养宠心里需求
小慧是一个在校的大学生，特别喜欢宠物，但是由于目前的条件限制，不能完成这个心愿
，所以希望能够先学习养育的经验，而后在有条件的时候在进行养育。
有一天小慧了解到宠趣APP，在这里可以浏览各种品类的萌宠视频，针对将来养育的宠物
品种中查看对应的养育帖子，看其他宠物分享，满足自己希望养宠的心理安慰。
小结
（1）对于年轻用户，产品视觉要时尚个性，色彩鲜明轻快，有差异化、趣味化。
（2）对于有购买需求用户，产品主打品牌产品为主，产品上可以分为高档和中档产品，
要保证产品质量，同时在商品评价板块中可以查看看相关的关联帖子，让用户更加了解产
品情况。
（3）社交圈子板块分为以下大类：养宠经验交流类；宠主晒日常类；产品使用心得类；
知名养宠达人专题类；养宠知识百科。

（4）视频板块可以考虑用户直播的可行性。
（5）打赏机制，考虑用户可以针对其他优质内容进行打赏，优质内容的创造者可以提现
。
（6）考虑引入附近，查看附件的宠物医院，相关宠友点评等。
四、竞品分析
4.1 竞品选择
此次竞品分析的对象主要是选择于“宠物电商用品”类应用，同时选择对标的应用有2个
，分别是：“波奇宠物”、“E宠商城”。
4.2 竞品分析
产品战略定位


（1）波奇宠物APP
波奇宠物APP的Slogan是“全面关爱宠物生活”，可以看出：波奇宠物定位是多方面满
足宠物用户群体需求。
波奇宠物以论坛用户内容为主打，商城销售为盈利点，线下服务为辅助，实现线上线下社
交为一体的宠物平台。
目前波奇宠物的优势在于：重视用户创造内容，通过论坛交流学习分享养育经验，在线上
选择商品购物同时还可以在线下选择店铺服务。在这个层面上，启发我们初期应做好产品
核心功能的架构和深化。
（2）E宠商城APP
E宠商城APP的Slogan是“正品、精选、简单”，可以看出：波奇宠物定位是提供品质
宠物用品购物电商平台。
E宠商城以提供高品质的商城购物，以专业的用户创造内容为引导，让用户享受便捷的购
物体验。
重视商城板块，页面设计与交互给人耳目一新的感觉，专业的体验测评师，让用户更好的
去了解产品，从而转化商品更多的购买量。
 电商购物功能结构分析
（1）波奇宠物APP

对于电商类APP而言，购物功能是产品最基本的功能，也是核心的功能。从购物以及支
付的流程上分析，与常规的电商购物平台上无特别大的差异。
波奇宠物在购物订单时效性为1个半小时，在购物时、结算时均有提示是否开VIP年卡，
以及所能享受优惠，可以提供商城VIP的购买转化率，提高用户再次选择波奇宠物商城的
购买率。
（2）E宠商城APP


E宠的购物流程恕我直言有点非主流，与常规用户所熟悉的购物流程中缺少了关键的立即
购买，只有加入购物车然后去购物车中支付。
就站着用户角度而言，立即购买是对该商品有着强烈的购买欲望，加入购物车是对该商品
有想购买的欲望。反用户习惯的操作是为了让用户多点时间思考是否要买这个商品吗？
社交功能结构分析
（1）波奇宠物APP


波奇宠物在社交板块比较看重，用户可以创造内容，以图片、文字、视频的方式进行分享
，还有有问答百科的形式丰富用户经验，期间穿插的活动内容增加用户粘度。
波奇宠物在社交上可增加对内容创造者、礼物打赏机制，可以给用户更大的动力。鼓励用
户分享波奇商城产品体验内容增加商品流量。问答板块是否可以聘请专业兽医进行解答进
而加入宠物医药板块。
（2）E宠商城APP
E宠商城上仅有PCG专业生产内容，用户可以浏览内容的同时购买内容中的商品，用户
可以在内容下留言、评论、收藏、点赞。
E宠商城的PCG板块，浏览量和评论量相差较大，从这里可以看出用户对于过于商业化
的内容还是比较反感的。可以鼓励用户发布上传自己已购买的商品测评体验，保证了一定
的真实性。


小结：
E宠商城APP代表了为宠物电商平台的注重商城板块的产品，而波奇宠物APP代表了社
交+商城+线下服务的精品产品。
把这两个产品作为宠趣APP竞品，可以分别学习这两类产品的优点和经验，同时关它们
的缺点和不足，以便使宠趣APP扬长避短，更好地完善产品体验。
