# 产品需求文档：英语流利说，教你流利说英语

编辑导语：英语作为一种国际化语言，一直以来都是各国学习的重点。然而在学习基础的
理论之外，能够真正地与外国人进行交流也是很重要的，于是出现了越来越多的能够提高
口语能力的英语APP。其中，就包括英语流利说。
一、文档概述
1. 文档属性及写作目的
本文档主要是对英语流利说业务需求进行收集、整理和分析，本文档描述了该软件
V8.18.0所要实现功能的详细描述和要求。为了达到便于理解的效果，文档中使用了相关
必要的和规范的形式对需求进行描述。例如，业务流程图、用户操作页面等形式。
文档供用户方和开发方的相关业务人员、设计人员、开发人员的使用，“英语流利说”是
一款融合创新口语教学理念和尖端语音评估技术的英语口语学习应用，一款旨在让你开口
说英语的软件，目标让你忍不住开口说英语。
笔者通过体验英语流利说免费及付费功能，倒推了英语流利说的需求文档PRD，目的是
练习Axure软件，了解产品需求文档的写作流程和大致模块。


2. 项目背景
1）行业分析
据艾瑞咨询统计核算，2020年中国在线教育行业市场规模2573亿元，过去4年的
CAGR达34.5%，其中低幼及素质教育赛道、K12学科培训赛道在线化进程加快是在线教
育市场快速增长的最主要贡献因素。
疫情影响下，低幼及素质教育领域的在线化范围持续纵深，职业教育领域的在线化进程也
在不断加速，新的供给和需求不断产生。预计2025年中国在线教育市场规模将达到
8000亿元，2020到2025年复合增长率达11.4%。
2）产品定位
“英语流利说”其slogan为“让你忍不住开口说英语”，由此可以看出其定位是帮助用
户实现流利的说英语。其借助于人工智能技术为用户提供地道科学的口语练习，帮助用户
提升英语听说水平。
3）产品特色
不同于其他在线教育软件，英语流利说的特色在于AI教学，随到随学，不受时间地点的
束缚。少了真人教学的尴尬，面对机器教学打分，让更多羞于开口表达的学生更大胆的开
口说英语。借助精确的智能系统打分可以帮助系统认知自己的发音不足，也可以通过评分
看到自己的进步。

四、产品详细功能说明
1. 登录注册
登录方式有多种选择：
1）本机号一键登录
自动识别本机号码，一键登录，如果是已经登录过的账号则按步骤进入app界面。如果
系统识别到是未注册过的手机号则自动注册生成一个账号，避免用户多次操作途中流失。
2）短信验证码登录
登录手机号大于等于4位数字。
手机号输入交互说明：输入框默认状态显示文本“点击输入手机号”，点击输入框弹出数
字键盘；输入数字后，显示清空数字按钮，点击后清空当前所输入数字内容。
验证码格式：验证码为6位数字。
限制条件：登录验证码有时间和数量限制，超过后数量限制后toast提示“操作过于频繁
”；用户收到验证码并在5分钟内输入即可登陆，超过5分钟验证码失效需要重新获取。
验证码格式不正确或为空（前端校验）： toast提示“验证码错误，请重新输入”。
验证码交互说明：输入框默认显示“输入短信验证码”，点击弹出数字键盘；手机号为空
时，点击“发送验证码” toast提示“请输入手机号”；点击“发送验证码”按钮，前端
验证手机号格式是否正确，通过验证则系统发送验证码，未通过验证则toast提示“请输
入正确的手机号”。
3）其他第三方账号登录
第三方账号登录有以下几种方式：手机品牌账号ID、微信、QQ、新浪微博、邮箱；登录
规则遵循已登录过的账号自动识别，未登录的账号自动注册生成一个账号。

分析：流利说登录注册采用了简化操作的流程，能简单的步骤绝不复杂来做。对用户来说
能减少一步操作绝不分两步来做。登陆注册便捷化的好处是避免用户在多步骤操作过程中
流失，带给用户简洁易用的感受。降低了注册门槛使游客到注册用户的转化率变高。
但这样会产生几个问题：一是同一个用户可能会无意识的注册多个账号，有时会忘记自己
的主账号是哪一个，后续可能需要用户手动绑定手机号。二是注册门槛降低会使注册用户
变多但相对来说忠诚用户占比会降低。
2. 引导页
引导页出现在用户首次登录注册流利说app时，引导页的目的是了解用户基本信息，为
后续个性化定制提供数据。

分析：引导页相当于用户与app初次见面的打招呼，因此界面中也展示了一个流利说AI
的IP形象，用谈话的方式和口吻采集用户信息，操作步骤简单，给用户亲切易用的感受
，同时也不忘引导用户体验流利说独有的AI智能水平测试功能和尝试给用户推荐付费精
品课。
由此可见，流利说的每一个界面和环节都是服务于付费用户转化的。
3. 学习页
学习是流利说app的主要功能，因此放在首页的位置，首页主要分为三大块内容：
1）学习时间及任务卡片
由用户是否购买付费课来决定学习时间卡片的样式。若用户未购买付费课，则卡片内容以
记录轻松学课程任务，学习时间，打卡天数为主；若用户已购买付费课程，则以精品课学
习时间和学习天数为主。
每日进行学习，系统自动打卡，学习时间满30分钟高亮显示打卡，有学习但未达到30
分钟打卡图标显示浅色，未学习则不打卡。
2）付费精品课入口
首页点击精品课跳转至课程地图； 课程地图以地图的形式展示目前用户所处课程章节的
位置，并以用户头像的方式展示用户所处进度附近有哪些用户，一方面让用户看到学习路
上有其他同伴，另一方面可以此刺激用户坚持学习； 点击地图下方开始学习按钮进入课
程详情，课程详情描述了本节课的主要内容和重点； 点击页面下方开始学习按钮正式进
入课程学习内容。课程根据用户的水平智能设置对应的知识点和练习； 小节课程学习结
束后会有学习评分反馈，用户可以直观的看到自己的成绩，对比每次成绩评分可以看到自
己的进步； 评分反馈页面设置有对课程的满意度反馈或学习期望，以便于AI智能安排下
次学习的内容。
3）免费轻松学课程入口
首页点击课程列表里的轻松学课程跳转到课程详情 点击课程标题进入正式课程学习 课程
结束后智能评分

分析：精品课是流利说的重点功能，也是营收的主要板块。在课程设置上配合英语水平测
试，制定了更加符合用户当前水平的课程内容。并且每天的课程并不是一成不变的，而是
根据用户上一节课的学习情况和用户反馈来定制。课内的题型也是多样性，听说读写四个
方面的能力兼顾。
每次练习错误2次以上就会弹出相应的题目解析，发音不标准的单词会用红色显示，发音
题会有评分。用户做题可以有非常好的反馈效果。
4. 提升计划页
此页面是首次进入app的引导流程的后置页面。此页的展示情况分为三种：
1）未参加英语水平测试，或已参与水平测试，或手动设置等级水平，但未设置学习目标
的情况
此情况下提升页面显示“继续测试”按钮，点击按钮继续测试，或手动设置自己的英语水
平，完成学习目标设置。
2）已参与水平测试或设置等级水平，同时也设置了学习目标，但未购买课程
此情况下提升页面显示“设定学习目标”按钮，点击按钮设置学习目标，界面跳转至懂你
英语落地页，如果不购买课程退出，此页面依然显示“设定学习目标”按钮，并且下次进
入时需重新设置学习目标。
3）已购买付费课程
已购买付费课程，此页面显示“去学习”按钮，页面跳转到学习地图。

六、总结
体验过英语流利说该版本之后，感受是有些地方的布局略显繁琐，例如“懂你英语工具卡
片”目前放在“我的”页面中，“卡片中”包含课本，错题收藏等，但懂你英语的精品课
程学习入口在首页，营销引导页后跳转到“提升计划页面”，如果将懂你英语“工具卡片
”归类入“提升计划页”似乎更加合理。
发现页中多处点击跳转都指向“懂你英语精品付费课”，总观整个app中非常多的按钮
都会指向落地页，显得十分重复。发现页十分冗长，用户滑到最底端的可能性不大，分类
并没有十分明显的板块区别，如果能将检索分类更加明确和精简化会更好。
整个APP结构并不复杂，相对来说操作很简单，容易理解，也没有很深的层级，对用户
来说比较友好。课程设置的在此文档中没有详细写出来，但体验过后觉得十分智能和人性


化，会根据用户的学习程度智能调节下节课的内容，也支持用户手动更改智能AI配置的
课程计划。
写这篇文档时参考了各大平台各种帖子和模板资料等，有些具体的逻辑和规则比如排序规
则单从体验没办法得知，因此很多地方无法深入写作。一方面是资料有限，另一方面是自
己没有实战经验只能管中窥豹。
