二、 行业及市场分析
1. 移动阅读行业
广义的移动阅读，是指使用移动终端进行的所有阅读行为，包含通过浏览器浏览手机网站
，以及阅读新闻客户端、报纸客户端、杂志客户端、微博微信的文章等。
伴随着移动互联网的日益发展和日渐渗透，其所带来的红利逐渐显现，互联网大大改变了
人们的阅读方式，人们的目光也渐渐从线下实体书本转移到线上屏幕，而现在，移动阅读
俨然成为了当下中国用户阅读的主要渠道！
据易观报道称：2017移动阅读市场规模达到153.18亿元，同比增长29.16%，2018年
将是移动阅读高速发展期。
但当前阅读行业已逐渐成为几大巨头雄踞的竞争局面，阅文集团，掌阅科技，咪咕文学，
阿里文学作为第一梯队，且在2017年阅文和掌阅相继上市，竞争激烈。

2. 社交阅读分析
从2010年以来，社交网络风靡一时，各个细分行业都争相和社交挂上号，从社交视频到
社交电商、再到社交阅读，各行业参与者都希望为自己植入社交基因，从而带来更多的用
户和更高的利益回报。
就目前来说，市面上的社交阅读APP大都是基于小说和文学作品领域的内容，简单的评
论分享互动排名，而对于新闻媒体资讯这一垂直领域尚且没有较多的社交阅读平台可供用
户去阅读。
因为作为移动阅读中的细分领域，社交阅读的渗透率并不高，但个人认为也是有一定原因
的，客观原因便是大多平台缺乏一定的创新，运营手法老套，平台差异化并不高，用户粘
性不高。
其实，用户每天的信息获取来源主要是媒体资讯，而在当今这样的人人都是自媒体的大背
景下，用户被各种标题党的文章水货内容烦恼着，阅读积极性并不高，情况并不乐观！
而随着生活节奏的越来越快，时间成本也越来越高，用户对于市场面上海量的文章内容用
户没有时间进行适当筛选，导致低效率低质量阅读的问题异常严重，用户的阅读热情每况
愈下，每每跌入谷底！
现在，该问题已经逐渐凸显成为人们的一大痛点，甚至有可能上升为社会普遍的一大问题
，化解该痛点已是用户刚需，更是企业责任所在。
所以说，社交阅读市场在资讯内容领域有望寻求一个创新突破，寻求新的平衡点来抵制当
前范媒体的大环境给人们阅读带来的困扰！所以说该市场潜力很大！
3. 社交阅读市场
随着生活节奏的不断加快，越来越多的人不断出现知识焦虑的现象，他们渴望从海量知识
里面寻求“黄金屋”以便慰藉。
因为“人是社会动物”，社交已然是每个人生命中不可或缺的一部分，在阅读上也是如此
，需要社交来带动它，驱动它向前。

随着移动用户的日益庞大，移动阅读平台不断在增多，用户对于阅读的社交需求日益凸显
。据亚马逊发布的：“2017全民阅读报告”显示，78%的受访者选择通过社交平台分享
与阅读有关的内容，与网友“一起读”成为数字阅读时代的独特行为，可见阅读社交化日
益明显，社交阅读是大趋势！阅读低效率低质量地热情问题突出。

四、 竞品分析
1. 竞品概述
严格意义上说：“布氪”本质上是一款媒体资讯类程序，它的竞品并不是像“微信读书”
或者“掌阅iReader”这一类的图书文学阅读应用，其是耕植于新闻媒体资讯这一细分领
域的。


用户可利用自己的碎片化时间来阅读资讯干货进行消遣，提升自我，缓解知识焦虑。所以
它与市面上的新闻阅读资讯类产品构成竞品关系，尤其是Nuzzel类的聚合阅读器应用，
将形成直接的竞争关系！
而其中构成直接竞品关系的应用，据本人浅陋的了解当属 轻芒阅读，ZAKER新闻和
Flipboard红板报了！
2. 竞品分析
轻芒阅读是一款兴趣类资讯阅读类应用，它的用户群体是对互联网较熟悉的年轻人，用户
可自主选择兴趣主题或应用，而后应用将从其他内容源获得到相关主题的内容提供给用户
阅读。简单来说就是一个第三方内容获取入口,体验很好的阅读器，很轻量。 ZAKER阅读
app内容质量高，广告少，用户年龄集中在19-45岁的年轻群体，且粘性较高。值得一
提的是ZAKER有自营商城，能让用户的阅读成果在一定程度上得到变现。 Flipboard红
板报算是国内资讯阅读类软件的鼻祖了，全新改版的它实现了真正产品本土化，符合中国
人的阅读习惯，自认为内容feed非常赞，简洁大方；海量话题发现，个性化内容随性选
择，用户体验蛮不错。
3. 竞品结构
（1）轻芒阅读APP结构
轻芒阅读在这些应用中的功能还算是蛮基础的，而非常重要的一点便是杂志式页面，交互
等细节做得挺好，用户体验的前中后期都较不错；但用户无法评论留言深入交流，给人更
大的感觉便是RSS阅读器，一个第三方的阅读工具，阅读后期的服务有待提高。


（2）ZAKER新闻APP结构
ZAKER新闻是一款颇具用户欢迎的个性化定制应用，用户群体主要集中在中高端用户，
互动分享功能在这些应用中很抢眼，每个用户都有自己的社区，可分享交流。
但应用整体功能稍多，除了基本的订阅阅读功能外，应用还有直播、视频、话题、动态等
功能，操作有点复杂，可能会让用户产生一种困惑的感觉。


（3）Flipboard红板报APP结构
Flipboard红板报创新式的阅读交互界面让人眼前一亮，让人情不自禁想要多刷一番，独
有的国外内容来源是加分项。


4. 总结
从以上产品结构图，可以看出各个阅读产品现在都已经相当成熟了，都具有极其相似的运
行模式，他们都是聚合类阅读平台，集合各方应用的内容，而且各自拥有自己的细分市场
和用户群体。
这三款产品普遍具有很多相似的特征，诸如用户可根据自身兴趣爱好自主订阅来自如微信
公众号&资讯应用程序&网站等等的资讯内容。用户在一个应用内即可阅读浏览到丰富的
内容资讯，操作简单，高质高效，且用户体验良好，每个应用都相当不错。但这些应用却
一直不温不火，一直被拥有高技术壁垒的头条所打压。
而在这个一超（头条凌压）多强、巨头林立的资讯阅读行业中，新品想要破土而出并成长
具有一点姿色确是很难。
但也并非无机会，只要在垂直领域做市场纵深，只要在细节处做好做佳，只要有另辟蹊径
，以创新的阅读模式呈现给用户，给予用户新颖的更加良好的阅读体验，打破现在已有的
阅读市场形势并非无可能！布氪便是如此，人无我独有！
五、产品概述
1. 产品简介
布氪阅读平台是一款集内容分发阅读、布氪创作共享、互动分享等功能为一体的阅读平台
，阅读模式新颖，其所瞄准的细分阅读市场是社交阅读！
当前市面上已有的阅读平台尚未能为读者搭建起能够达成共识的朋友圈子，阅读模式一贯
如此，没有创新点，不能很好满足当代年轻人的新型社交阅读需求。在同一个场景内达成
共识并完成思想上的碰撞体验，这是“布氪”平台现在要做的，也是平台的希冀所在！
2. 产品定位
一个能够满足当代年轻人多样化阅读需求的内容平台，用户在内容中达成共识，在平台上
完成体验！布氪，不可思议！
3. 目标用户群体
目标用户：主要位于一二三线城市，年龄在18-40岁左右的那些熟悉互联网且喜欢阅读并
对现有阅读方式产生困倦的年轻人。

目标市场：线上阅读市场中的细分领域，社交阅读市场！
