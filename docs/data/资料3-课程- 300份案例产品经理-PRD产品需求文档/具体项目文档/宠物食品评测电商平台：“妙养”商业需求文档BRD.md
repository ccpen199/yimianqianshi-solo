# 宠物食品评测电商平台：“妙养”商业需求文档BRD

中国宠物市场发展飞快，宠物食品竞争加剧。笔者通过分析国内宠物市场，提出了对宠物
食品评测电商发展趋势的思考。
一、项目背景
1.1 需求来源
随着我国国民收入的提高，老龄化以及低生育率等问题的加剧，越来越多可爱的宠物走进
了千家万户，成为了人们重要的陪伴与情感寄托。
但由于我国宠物市场起步较晚，发展不完善，再加上政策与监管不到位，出现了很多问题
。其中，比较突出就是宠物食品安全问题。
例如，2016年湖南警方查处的宁乡县一个制造假猫粮狗粮的窝点，查获了60吨假宠粮。
据悉，有13家淘宝网店同步售卖这些“毒粮”，甚至销量和排名十分靠前！并且假猫粮
中大部分还都是仿冒的“进口粮”，真是可恨！
虽然淘宝也在加强打假，但买到假猫粮狗粮的新闻依然层出不穷。就连垂直电商波奇网、
E宠商城也未能幸免，接连出现“真假混卖”、“有托”的爆料。
而归根到底，还是这些无良商家知道，大部分的宠物主人不具备辨别真假的能力，他们利
用人们贪图便宜的心理，很容易得手，赚取暴利。


近期，上海市消费者权益保护委员会进行了一次狗粮抽样大检查。选取了市面上常见的
48款狗粮，针对狗粮是否受到污染、毒素是否超标、卫生是否达标、营养成分是否虚标
等进行了详细的检测。
结果大量本土品牌被检测出问题来，检测结果显示，有的品牌产品玉米赤霉烯酮含量超标
，有的品牌产品菌落总数超标，有的品牌产品营养成分含量虚标，有的品牌产品还涉嫌虚
假宣传。
中国宠物市场发展飞快，宠物食品竞争加剧，细分个性化的产品越来越多，无疑会有很多
新的原料和配方出现。一方面造成了监管的压力，对宠物主人来说自身也需要越来越专业
。
根据“微博话题”和知乎问答上的统计，关于如何选购猫粮与狗粮，有超过2000万余次
的关注。可见很多人在选购宠物粮食的时候，遇到了很多困扰。


同时也可以注意到，食品安全只是最基本的要求，宠物们吃得更加健康，吃得更加营养，
对宠物主人来说已经越来越重要。
1.2 用户需求
宠物主人在选购宠物粮食的时候会遇到这样的问题：
分辨不出来哪些是真的，哪些是仿冒的； 区分不出来品质的优劣，只能以价格及其他人
的评价作为判断依据； 缺乏必要的宠物知识，不了解宠物粮食的营养成分，不知道自己
的宠物应该吃哪种； 对宠物零食、宠物保健品等新产品一知半解； 大包装宠物粮食重量
可达到20公斤，每次扛到家里都太痛苦了；
分析宠物主人选购宠物粮食遇到的问题，可以发现用户的本质需求是方便快捷地买到放心
并且适合自己家宠物的宠物粮食，包括零食、保健品等，最好还可以送货上门。
1.3 市场背景
前面提到因为经济发展，老龄化以及单身人口等问题的加剧，我国宠物经济市场进入了飞
速发展的阶段。

根据《中国宠物行业白皮书》的数据显示，自2010年到2018年，中国的宠物行业市场
规模由140亿元增长至1708亿元，8年增长了约12倍，年均复合增速在36.7%，有望
在2020年突破2200亿元。


目前我国已经形成了涵盖上游：宠物饲养、活体交易；中游：宠物食品、宠物用品；下游
：宠物培训、宠物寄养、宠物保险等一整条的宠物产业链。
而纵观整条产业链，宠物食品是最为刚需，占比也是最大的一部分。


根据《2017年宠物行业白皮书》的数据显示，宠物商品类消费额占宠物总体消费的70%
，而宠物食品，包括主粮、零食和保健品的消费额在商品类消费额中达到了80%。
可见，宠物食品在我国整个宠物消费中举足轻重的作用。


根据数据，2018年宠物食品市场超过500亿元。


细分宠物食品的销售渠道我们发现，线上渠道的占比正在逐年扩大，目前已经成为了最大
的渠道。网购成为了很多宠物主人采购宠物食品的首选方式。
目前我国宠物发展还处于起步阶段，人均宠物拥有数量少，单只宠物消费不足，导致了我
国宠物消费在地理位置上比较分散。一个规模较大社区中的猫狗数量才能够支撑一家线下
宠物专业门店的经营。
并且线下门店规模一般较小，以美容、洗澡、寄养等服务为主，能够留给宠物食品销售的
空间有限。这就导致了线下宠物门店中产品的品牌、种类单一，宠物主人的可选择产品范
围很小。
而宠物食品进驻商超等零售渠道也会受到入场费、品牌知名度等限制，并且线下商超自身
就在承受来自线上的竞争压力，对宠物食品行业的发展很难有加成。


在这种情况下，电商成为新品牌快速放量最具优势的销售渠道。一方面是国民对网上购物
的接受程度越来越高，电商平台把分散的宠物食品品牌集中起来了，相比于线下分散的门
店，降低了宠物主人的搜寻成本。
电商利用我国宠物总体数量大但低密度的特点，通过网络这个“场”拉近了“人”与“货
”的距离；
而拼多多等主打下沉市场的电商平台，则进一步降低了用户的使用门槛，也许第一次网购
就是拼了一袋狗粮。
1.4 竞争格局
聚焦宠物食品的线上销售渠道，国内主要是以淘宝/天猫、京东、拼多多为代表的大型综
合电商平台，波奇网、E宠商城为代表的宠物垂直电商平台。
数据显示，2018年阿里系宠物食品类目销售额达到了100亿元，占据了线上渠道大部分
的蛋糕。而相比这下，垂直电商由于流量弱势，份额遭到不断挤压。


所以，这些垂直电商也会选择在淘宝、京东上开店，借助大平台的流量优势，来增加销售
额。
根据公开资料显示，在2018年双十一大促活动，波奇网全平台成交额突破1.6亿，较
2017年全平台1亿元同比增长60%。
其中天猫波奇网旗舰店销售额突破8030万元，较2017年的4875万元同比增长64.72%
，在天猫宠物类目排名第一。而E宠2C零售全体系销售额也突破了1亿。
成立于2008年初的波奇网，起步于论坛社区，之后进入宠物电商领域，在2014年又进
军宠物服务领域；经过数年发展，波奇网既是目前国内规模最大的宠物电商，宠物零售渠
道，以及98%在合作供应商最大的销售渠道，也是国内最大的宠物服务提供商。
而同样成立于2008年的易宠科技，是一家专注于宠物行业的互联网公司。并在2009年
上线了宠物用品独立B2C平台“E宠商城”。
2014年，E宠商城开始学习京东，探索智能化的仓储物流管理，开始建立起自营物流团
队。有人戏称之为“宠物界的京东”。E宠商城在2014年，全年销售额突破一亿元。
为什么宠物垂直电商发展到现在，已经过去10多年了，面对近千亿级的市场规模，还是
没有巨头公司出现呢？
一方面是在国内养宠观念虽然有了大转变，但是宠物主人在喂养宠物时，大多都缺乏经验
，还未达到科学喂养的标准，这很大程度上限制了宠物垂直电商的发展；
另一方面是宠物电商同质化严重，产品与服务没有形成差异化，不能树立口碑。进而造成
了宠物主人对品牌的认知度低，忠诚度低。在选购宠物食品的时候，相较于大型综合电商
平台的品牌与服务优势，宠物主人更愿意选择后者。
二、项目规划
2.1 核心产品需求
基于评测内容的垂直宠物电商平台。

2.2 产品结构
基于产品的核心需求，我们设计有评测内容、申请评测、问答社区、商城、消息通知和我
的等功能模块。
首先我们要立足提供高品质的宠物食品评测报告，通过视频或者图文结合的方式，展示对
一款食品的全面解读，力图让用户在这里通过我们的报告能够清楚地了解某一款食品。
根据用户对评测报告的反馈情况，我们可以考虑增加购买渠道。
围绕宠物食品，可以延伸出来养宠物的多种问题，所以增加问题或者话题功能模块，让在
饲养宠物时候遇到问题的用户有发问和获得答案的快速方式，也可以增加用户的使用黏性
。
