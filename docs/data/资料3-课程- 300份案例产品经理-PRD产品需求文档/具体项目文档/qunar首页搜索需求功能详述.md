页面属性：
	•	单程/往返：单选项，必选，默认选择单程；
	•	运营活动链接：可点击项，绿色字体；
	•	出发输入框：必填项，默认为当前访问IP对应的城市；
	•	到达输入框：非必填项；
	•	出发日期：必填项，默认为后天；
	•	返回日期：选择单程时，该项默认为空，可点击；选择往返时，该项默认为后天+3天；
	•	可实时搜索N条国内国际航线：N为当天可飞行的在线航班数；
	•	立即搜索：按键，可点击；
