6.1 登录注册
6.1.1 页面流程图
6.1.2 页面名称：登录注册页面
6.1.3 页面入口
未登录状态下，用户点击喜欢、收藏、投食、关注、弹幕、评论、聊天、礼物、购买、我
听自动进入登录页面，打开特色功能，如全职高手角色语音包、微信男友也会进入登录页
面。
6.1.4 前置条件：网络连接正常
6.1.5 逻辑与交互说明
手机号/邮箱密码登录：在输入框内输入账号密码，系统判断该账号类型为邮箱还是手机
号，同时对不合法手机号进行提示，账号密码均正确则登录成功，进入“我的”页面。 


海外手机号登录：选择海外手机号登录，选择国家或地区，之后流程同手机号登录。 第
三方授权登录：猫耳FM提供的第三方登录方式包括哔哩哔哩、QQ、微信和微博，点击
任意第三方按键，跳转至第三方授权页面，非首次使用该方式，则自动跳转登录成功。首
次使用该方式时，同意授权后需绑定手机号，若该手机号为已注册账号，则登录成功，进
入“我的”页面；若该手机号为新账号，则进入个人资料编写页面，需输入昵称且设置密
码，之后进入“我的”页面。 立即注册：点击“立即注册”，进入注册页面，选择国家/
地区，输入手机号，点击获取验证码，系统判定该手机号是否合法，对不合法手机号进行
提示，合法手机号判定是否为已注册账号，已注册则提示”该账号已注册“，未注册则发
送验证码，验证成功后需设置昵称及密码，完成后进入”我的“页面，可在此页面中绑定
邮箱，之后使用邮箱密码登录方式。 忘记密码：点击“忘记密码“，进入找回密码页面
，输入手机号或邮箱，系统判定该账号是否为注册账号，是则发送验证码，验证成功后可
修改密码；若一般方式不可用，可选择人工申诉重新设置密码。
进入子页面只能选择点击，但从子页面返回可以右滑。
6.1.6 优化建议
当前版本（ios4.5.5）的注册登录流程过于繁琐，且注册与登录是分开的，建议
将注册模块取消，直接未注册用户登录即自动创建账号； 添加手机号验证码登录方式。

7. 总结
猫耳FM是一款正在成长的产品，在二次元文化日益繁荣的现在，拥有庞大的IP改编和
声优粉丝，专注于声音领域， 猫耳已经抢占了先机，但依旧有需要完善的地方（来自用
户体验以及七麦数据用户评论）：
登录即注册功能：不能实现手机号验证码一键登录，未注册用户登录即自动创建账号的功
能（最新版本已解决）； 前进后退功能：音频播放中会出现经典台词或经典片段，不能
设置前进后退15s，用户自行拖拽进度条很麻烦； 跳过片头片尾功能：在放松或入睡时收
听剧集时，突然响起的片头片尾声音很大； 直播回放功能：希望能够添加直播回放的功
能，当前只能声优投稿直播音频，不能回放直播。
