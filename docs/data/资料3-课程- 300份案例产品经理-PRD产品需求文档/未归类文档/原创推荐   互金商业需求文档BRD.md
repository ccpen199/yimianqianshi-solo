# 原创推荐   互金商业需求文档BRD

原创推荐  | 互金商业需求文档 BRD  
最近几年 P2P行业异常火爆。根据相关数据，截止到今年 10月，全国的 P2P平台数量已
经高达2800多家了。在如此火爆的市场下，竞争也异常激烈，各家有各家的特色。  
笔者也曾在 P2P第三方公司任职，期间参与了多个 P2P平台设计的项目，从车贷、房贷，
到票据、信贷，在做项目的过程中学习到了这个行业的各种发展思路。今天把这段时间积
累的东西做一个总结和归纳。  
 
 
本篇介绍的 POP平台是基于当前市场大背景下的一种全新平台模式的探索，该模式下的平
台更像是一个各类 P2P的集合体，能满足小平台和大平台的各类需求。本篇商业需求文档
针对项目本身的可行性和成本预算做了整体的概括。 PPT做的比较精简，表达出大致的思
路和想法。
