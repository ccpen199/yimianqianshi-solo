# 面试通产品需求文档

编辑导读：互联网时代，最重要的就是信息差。但是，在信息技术这么发达的当下，信息
泛滥，反而让人们无所适从。尤其是对于求职者来说，精准的求职信息对他们来说至关重
要。今天，我们就针对这一强烈的需求，设计一个基于求职面试的全生命周期的APP。本
文就是与之相关的产品需求文档，我们一起来看看。
影响互联网求职的重要核心要素之一在于信息差，学笔试有牛客网，而面试却没有一个专
门的APP。
基于此，笔者想设计一个基于求职面试的全生命周期的APP，以供求职者搭建一个社区，
学习知识、积攒经验并练习面试。

二、产品介绍
1. 产品定位
本产品面向互联网行业求职者，笔者针对应届生求职期间忙、乱、杂、慌这一痛点，设计
一款管理用户求职全生命周期的APP，为求职者们搭建一个精准交流的平台。
使用此APP，用户可随时学习、突击训练、获取前辈经验、规划求职路线。


APP主打UGC模式，鼓励用户积极发帖发问积攒经验，为大家创造一个更公开活跃的互
联网求职环境。

3. 需求分析
1）互联网求职人数众多
从年龄分布来看，18-25岁的年轻人更偏好选择信息技术服务业、文化娱乐业和金融业。
这届年轻人伴随互联网的发展而成长，见证高科技技术带来的创新产品改变人们的生活，
也在中国文化产业迅速发展的时代吸收更多的精神内涵。
行业对年轻人的影响力十分显著，对年轻人有足够的吸引力。
数据来源《艾瑞：年轻人求职目标朝多元化发展—“超职季”观察·年轻人篇》
