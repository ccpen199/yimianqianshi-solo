产品开发目的
以母婴为切入点解决用户的社交需求，增强用户粘度；高效快捷解决孕妇网上挂号难、健康、上门服务、专业知识、紧急情况的血源需求等棘手情况，从而解决用户的根本问题。
2.利用移动互联网的新技术制定出一系列在家就可以进行简单产检的功能，来解决一些孕妇产检方面的难题。
3.顺应公司发展战略，PC端产品发展成熟，随着移动互联网发展，移动端开发势在必行；当前母婴产品的信息更多的是以文章、论坛帖子这样的形式存在，缺乏相对完善、便捷的数据化母婴服务。
　　传统运营模式下，一个母婴产品的用户如果遇到问题，要么需要海量的扫贴来自行汇总信息，要么求助于几个所谓的社区公知，再有就是咨询身边熟悉的有经验的人。而移动互联网的用户行为本质却是碎片化时间参与和实时交互信息。
　　如此看来，以社区论坛形式承载的妈妈用户的实时互动应用，以其发帖便利性、信息获取即时性、多样性和个体精准性，得到了更多妈妈用户的青睐。

4.通过一些活动来帮助一些品牌进行品牌的推广，从而积攒用户和投资的积累。打造属于我们自己的品牌效应。符合现存用户需求，抢占竞品用户，挖掘潜在用户。

2. 问题
2.1 描述用户的“切肤之痛”
1.随着经济的增长和城市化进程的加速，准妈妈们的社交得不到很好的满足，社交的安全性有待考虑；
2.因教育程度不够或经济基础较低的用户们，得不到一些比较专业的孕产知识的资讯和护理；
3.挂号难，就诊难，经验不足
4.遇到一些突发的情况態及时解决
2.2 简介目前行业是如何应对这些问题的
就目前行业而言，有很多款母婴类APP软件，为满足妈妈们的社交需求，以及孕妇医疗等问题，开发了基于LBS定位的医疗服务功能和社区讨论功能，让妈妈们可以在一起讨论心得或者育儿知识。
3. 解决方案
3.1 阐述公司的产品/服务的价值定位如何解决客户的难题
面对我们上面所述4项难题，孕妈为用户提供了基于LBS定位的社交功能、O2O模式的宠物服务功能，以及寄养实时监控和商户闲置狗狗租赁这两大亮点功能，配以摇一摇就近配种提高效率和互动性，为用户提供更放心、更便捷的服务。

8.5 产品/服务的开发计划
8.5.1基础业务建设期
该阶段属于平台及基础业务建设期，主要以平台建设及基础业务内容和资源整合为主，为后续发展用户打好基础。该阶段以扩大产品影响力和扩大商户规模为主。
在产品平台运营过程中，基础业务建设阶段是最初也是最重要的环节。在该阶段需要通过与各行业合作的方式建设产品以此来推动整个产品项目的建设和发展。在建设的过程中我们需要达成三个目标。它们分别是：
1、收集业务、用户等商业信息、推动前期调研
2、获取相关内容，落实合作资源，推动业务上线
3、通过产品建设获取行业应用项目，打造标杆性信息化项目，提升产品的信息化收入
8.5.2精品应用建设期
通过打造产品精品业务，即社交、快捷医疗服务和专业指导三个部分，增强用户互动性和黏度，向用户提供高效快捷的服务，从根本解决用户的需求，快速扩大用户规模。
  8.5.3. 规模发展用户期
主要通过免费以及营销推广活动等，使业务迅速进入市场，聚集用户和商户群，提升知名度、提高使用率和使用量。该阶段以发展100万活跃用户为主要目标。
8.5.4. 盈利成熟期
 随着用户规模的不断扩大，吸引各类商户加入平台。借助产品平台，通过发布产品信息；帮助商户进行产品、服务的营销推广，获取销售反点、销售分成。实现线上社交、由前向收费模式向后向收费模式的转变，获得产品规模收益。
