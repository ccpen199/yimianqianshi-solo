名称
说明
猫扑联盟
通过一些机制，将猫扑的用户集合在一起的用户圈子，类似游戏里的公会。
联盟贡献
用以核定联盟升级的维度，不同的声望对应不同的联盟等级，联盟成立初始为100声望，提高声望到一定程度，可以自动升级。
联盟每天也会消耗一些声望，例如1级联盟每天减少10声望，2级联盟每天减少50声望，以此类推。所以，联盟如果长期声望不增长，联盟等级也会降低，换言之就是联盟不活跃的话会被系统降级！
用户贡献
用户通过发、回帖等各种行为，对联盟产生贡献。贡献值越高说明对联盟升级的贡献也越大。但联盟声望不等于联盟所有成员的贡献的简单加和。


	•	产品概述及目标
猫扑联盟系统本次优化，主要对现有功能进行了界面的优化，将大杂烩和贴贴2套联盟系统进行合并，合并后的联盟去除了某些功能。具体如下：
	•	用户和联盟的关系统一为一对多(一个用户最多可加入10个联盟)
	•	去掉了光暗阵营
	•	去掉了联盟印象、盟主令、联盟史传功能
	•	每个联盟不设置成员上限
	•	增加了联盟社交功能（小窝），已作为二期优化的重点功能。

二．联盟成员贡献度的的原则：
每给联盟增加一点声望，则记该成员给联盟做出一点贡献。由联盟成员发回帖和登陆两项指数决定。
1）、成员每发一帖，其联盟贡献度增加3点，每回一帖增加1点。其中发帖和回复数为审核之后的有效发帖和回复数。
发帖每被加为精华、热门帖、潜力、砖家帖，增加贡献度50点，被加为原创、神帖，增加贡献度80点。回复被加为精彩、神回复、213回复，增加贡献度10点。
